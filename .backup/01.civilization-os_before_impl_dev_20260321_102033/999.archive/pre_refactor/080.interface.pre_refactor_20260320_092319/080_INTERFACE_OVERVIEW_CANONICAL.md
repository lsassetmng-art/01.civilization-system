# ============================================================
# INTERFACE OVERVIEW
# ============================================================

status: canonical
layer: interface

Interfaces define the boundary between
CivilizationOS and external actors.

External actors include:

clients
persona systems
AI agents
games
streaming systems
external platforms

All communication must pass through
explicit interfaces.

Direct internal access is forbidden.

Interface types:

Event Interface
API Interface
Persona Interface
Client Interface
AI Interface
External System Interface

Interfaces must be:

versioned
documented
stable
secure
