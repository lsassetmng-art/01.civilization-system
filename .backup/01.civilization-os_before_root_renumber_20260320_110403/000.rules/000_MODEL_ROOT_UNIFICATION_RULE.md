## PURPOSE
Define the unified active model root for Civilization OS.

## RULE
- The active model root is 030.model only.
- Canonical operational models belong under 030.model/010.canonical.
- Expanded catalog models belong under 030.model/020.catalog.
- Binding documents belong under 030.model/030.binding.
- No other active top-level model root is allowed.

## NOTE
Former 050.model content must be absorbed into 030.model.
