# ============================================================
# 1. PURPOSE
# ============================================================

This document defines the UI architecture
for home and discovery in StreamingOS.

# ============================================================
# 2. UI GOAL
# ============================================================

The UI must support:
- home feed
- recommended content surfaces
- ranking-linked discovery
- category / topic browsing
- live-now discovery
- archive / clip discovery
- promoted or featured content visibility where allowed

# ============================================================
# 3. MAIN SCREEN BLOCKS
# ============================================================

Recommended blocks:
- top navigation bar
- personalized recommendation feed
- ranking-highlight panel
- live-now panel
- categories panel
- clips discovery panel
- archive discovery panel
- featured/promoted panel

# ============================================================
# 4. CANONICAL FIXED STATEMENT
# ============================================================

The discovery and home UI
shall provide viewer-facing discovery surfaces
for native StreamingOS content
without collapsing recommendation,
ranking,
and promotion into one opaque list.
