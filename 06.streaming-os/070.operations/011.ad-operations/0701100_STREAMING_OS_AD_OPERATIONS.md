# ============================================================
# 1. OPERATIONS PURPOSE
# ============================================================

This operations domain defines
day-to-day handling of internal ads.

# ============================================================
# 2. PRIMARY RESPONSIBILITIES
# ============================================================

Ad operations is responsible for:
- ad activation and deactivation
- insertion rule review
- ad burden/frequency oversight
- display-event review
- blocked or failed insertion review
- governance-restricted ad handling

# ============================================================
# 3. FREQUENCY OVERSIGHT
# ============================================================

Operations should confirm
that internal ad burden remains aligned
with YouTube-equivalent burden posture
as baseline reference.

# ============================================================
# 4. DISPLAY REVIEW
# ============================================================

Operations should review:
- shown events
- skipped events
- blocked events
- failed events

# ============================================================
# 5. CANONICAL FIXED STATEMENT
# ============================================================

Ad operations
shall supervise internal ad activation,
insertion control,
burden posture,
and display-event outcomes
without conflating internal ad behavior
with external platform advertising.
