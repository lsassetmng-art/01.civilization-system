# ============================================================
# 1. OPERATIONS PURPOSE
# ============================================================

This operations domain defines
day-to-day handling
for company-side streaming oversight.

# ============================================================
# 2. PRIMARY RESPONSIBILITIES
# ============================================================

Corporate channel operations is responsible for:
- supervising official company channel readiness
- supervising affiliated streamer visibility
- supervising company-owned stream follow-up
- supervising company-side governance/review issue visibility
- supervising company-side monetization visibility when relevant

# ============================================================
# 3. OWNERSHIP OVERSIGHT
# ============================================================

Operations should preserve:
- official company ownership meaning
- affiliated streamer participation meaning
- distinction between ownership and participation

# ============================================================
# 4. CANONICAL FIXED STATEMENT
# ============================================================

Corporate channel operations
shall supervise company-side channel and streamer oversight
without collapsing company ownership
into individual participation.
