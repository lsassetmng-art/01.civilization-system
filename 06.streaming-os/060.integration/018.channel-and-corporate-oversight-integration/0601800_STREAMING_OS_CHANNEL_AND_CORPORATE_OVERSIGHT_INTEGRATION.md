# 1. INTEGRATION PURPOSE
# ============================================================

This integration defines how channel identity
and corporate oversight contexts are consumed by StreamingOS.

# ============================================================
# 2. PRIMARY RESPONSIBILITIES
# ============================================================

The channel and corporate oversight integration is responsible for:
- consuming subject/company identity contexts
- consuming affiliation contexts
- exposing official vs affiliated visibility inputs
- supporting company-side oversight views
- preserving company ownership interpretation

# ============================================================
# 3. NON-RESPONSIBILITIES
# ============================================================

The integration does not replace:
- canonical channel records
- canonical corporate oversight records
- canonical affiliated streamer references
- canonical stream ownership records

# ============================================================
# 4. CANONICAL FIXED STATEMENT
# ============================================================

StreamingOS remains the canonical source
for channel-side streaming interpretation
and corporate oversight structures.

Integration may consume Civilization-side subject
and company affiliation contexts,
but may not replace StreamingOS ownership interpretation.
