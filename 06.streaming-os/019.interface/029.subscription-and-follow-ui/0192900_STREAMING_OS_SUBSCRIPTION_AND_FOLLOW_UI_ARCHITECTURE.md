# ============================================================
# 1. PURPOSE
# ============================================================

This document defines the UI architecture
for subscription and follow behavior in StreamingOS.

# ============================================================
# 2. UI GOAL
# ============================================================

The UI must support:
- subscribe / unsubscribe actions
- follow / unfollow actions
- followed channel list
- followed creator list
- follow status visibility on content screens
- notification preference entry where relevant

# ============================================================
# 3. MAIN SCREEN BLOCKS
# ============================================================

Recommended blocks:
- subscribe/follow action bar
- followed entities list
- follow status summary
- notification preference panel
- recommended follow panel

# ============================================================
# 4. CANONICAL FIXED STATEMENT
# ============================================================

The subscription and follow UI
shall expose explicit viewer-side subscription and follow behavior
as native StreamingOS platform features.
