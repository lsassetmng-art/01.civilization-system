# ============================================================
# 1. PURPOSE
# ============================================================

This document defines the UI architecture
for company-side streaming control in StreamingOS.

# ============================================================
# 2. UI GOAL
# ============================================================

The UI must support:
- corporate official channel management
- affiliated streamer list
- affiliated streamer status visibility
- company-owned stream oversight
- official vs affiliated interpretation visibility
- company-side monetization summary linkage visibility

# ============================================================
# 3. MAIN SCREEN BLOCKS
# ============================================================

Recommended blocks:
- official channel summary card
- affiliated streamer list
- company-owned stream list
- official / affiliated distinction panel
- revenue summary panel
- governance / review issue panel

# ============================================================
# 4. CANONICAL FIXED STATEMENT
# ============================================================

The corporate channel and affiliated streamer UI
shall expose company-side control
for official company streaming
and affiliated streamer visibility
without collapsing company ownership
into individual participation.
