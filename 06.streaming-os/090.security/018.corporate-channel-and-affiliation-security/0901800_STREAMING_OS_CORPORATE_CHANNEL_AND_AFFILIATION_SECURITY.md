# ============================================================
# 1. SECURITY PURPOSE
# ============================================================

This security domain defines how StreamingOS
protects company-side channel oversight
and affiliation structures.

# ============================================================
# 2. PRIMARY SECURITY OBJECTS
# ============================================================

Security focus includes:
- official company channel authority integrity
- affiliated streamer reference integrity
- company oversight visibility integrity
- company-owned stream control integrity

# ============================================================
# 3. OWNERSHIP DISTINCTION RULE
# ============================================================

Security must preserve the distinction between:
- official company ownership authority
- affiliated individual participation or visibility

Affiliation visibility must not grant company ownership mutation authority by default.

# ============================================================
# 4. CANONICAL FIXED STATEMENT
# ============================================================

StreamingOS shall protect official company channel structures
and affiliated streamer references
as explicit security responsibilities
without collapsing ownership into participation.
