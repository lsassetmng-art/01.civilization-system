# ============================================================
# 1. HELIOS
# ============================================================

internal_gap_notes:
- upper districts center on power-linked nightlife, inner-circle drinking,
  influence display, and face-network culture rather than refined etiquette alone
- worker districts center on taverns, nearby group-socializing, and practical recovery drinking culture
- outer districts are more cost-limited, rougher, and more dangerous in night-life form
- the same broad drinking culture exists across classes,
  but venue quality, safety, price, and political protection differ sharply


# ============================================================
# 2. NOVA
# ============================================================

internal_gap_notes:
- executive leisure is networked and premium
- ordinary urban leisure is commercial, compressed, and cost-sensitive
- nightlife districts generate distinct social patterns


# ============================================================
# 3. SEIWA
# ============================================================

internal_gap_notes:
- leisure gaps exist, but public festivals and family-compatible routines soften extremes
- community access remains comparatively broad


# ============================================================
# 4. GLADIA
# ============================================================

internal_gap_notes:
- service families have more structured shared leisure
- detached households have narrower and more functional recreation patterns


# ============================================================
# 5. ORPHEUS
# ============================================================

internal_gap_notes:
- island differences matter more than class-only difference alone
- ritual-rich and music-rich islands differ from quieter working islands


# ============================================================
# 6. AURELIA
# ============================================================

internal_gap_notes:
- protected groups and fragile households experience leisure almost as different worlds
- insecurity suppresses free time use in weaker districts


# ============================================================
# 7. FREE CITIES UNION
# ============================================================

internal_gap_notes:
- port, guild, civic, and market-centered cities create sharply different leisure ecologies
- local culture matters more than federal pattern


# ============================================================
# 8. FINAL RULE
# ============================================================

Internal leisure gaps must remain visible
so that national averages do not erase lived recreational difference.
