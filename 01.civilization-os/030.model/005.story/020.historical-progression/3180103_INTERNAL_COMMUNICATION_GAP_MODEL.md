# ============================================================
# 1. HELIOS
# ============================================================

internal_gap_notes:
- upper circles use dominance, familiarity, threat, and favor language more than refined formality
- worker districts use rough and direct speech with strong in-group codes
- outer districts show harsher, more volatile speech under insecurity
- the same roughness can signal power in upper groups and exposure in lower groups


# ============================================================
# 2. NOVA
# ============================================================

internal_gap_notes:
- executive speech is polished, strategic, and performative
- ordinary commercial speech is faster, looser, and deal-practical
- nightlife speech is more intimate, risky, and status-testing


# ============================================================
# 3. SEIWA
# ============================================================

internal_gap_notes:
- public norms reduce visible speech extremes
- school, office, and family each have clearer bounded registers
- status still matters, but control is formalized


# ============================================================
# 4. GLADIA
# ============================================================

internal_gap_notes:
- service families internalize command-sensitive speech more strongly
- detached civilians may sound looser, but institutional settings pull speech back toward discipline


# ============================================================
# 5. ORPHEUS
# ============================================================

internal_gap_notes:
- island variation matters more than formal class-only difference
- festival, fishery, brewing, and ritual islands each shape local conversational rhythm


# ============================================================
# 6. AURELIA
# ============================================================

internal_gap_notes:
- insiders speak with more doctrinal legitimacy and coded safety
- fragile outsiders may speak cautiously, indirectly, or with fear-heavy restraint


# ============================================================
# 7. FREE CITIES UNION
# ============================================================

internal_gap_notes:
- port, guild, market, and civic-discussion cities produce different speech ecologies
- local norms matter more than any federation-wide standard


# ============================================================
# 8. FINAL RULE
# ============================================================

Internal communication gaps must remain visible
so that national averages do not erase lived speech difference.
