# ============================================================
# 1. HELIOS
# ============================================================

ui_note:
- prestige and rough power aesthetics may coexist
- upper-class event presentation may emphasize status and patron presence
- lower-class event presentation may be harsher and more practical
- nightlife-linked tone may appear in some adult events


# ============================================================
# 2. NOVA
# ============================================================

ui_note:
- premium presentation, status polish, and branded visual logic are strong
- results may emphasize cost, prestige, and public visibility
- high-end and mass-market event skins should diverge clearly


# ============================================================
# 3. SEIWA
# ============================================================

ui_note:
- orderly, procedural, and institutionally clear presentation is strongest
- school, family, and civic ceremonial formatting should be stable and legible
- overdramatic presentation should be weaker than structured presentation


# ============================================================
# 4. GLADIA
# ============================================================

ui_note:
- duty, formality, rank, and disciplined presentation are strong
- military-adjacent life events may carry structured visual hierarchy
- wasteful ornament should be weaker than official order


# ============================================================
# 5. ORPHEUS
# ============================================================

ui_note:
- communal, musical, ritual, and island-local variation should be strong
- decorative expression may be warmer and more communal than formal
- family and festival feel should remain visible


# ============================================================
# 6. AURELIA
# ============================================================

ui_note:
- doctrinal tone, scarcity, and controlled presentation may dominate
- some events should visibly feel constrained, filtered, or austere
- protected insiders and fragile outsiders may need different presentation layers


# ============================================================
# 7. FREE CITIES UNION
# ============================================================

ui_note:
- strong city variation must be supported
- market, guild, civic, and port-style event skins may coexist
- local tone should often override federation-wide tone


# ============================================================
# 8. FINAL RULE
# ============================================================

Life Event UI must visibly change by nation
rather than only by event family.
