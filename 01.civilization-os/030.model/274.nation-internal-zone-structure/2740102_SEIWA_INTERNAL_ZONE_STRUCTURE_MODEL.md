# ============================================================
# 1. CORE SPATIAL RULE
# ============================================================

Seiwa developed from western mountains,
water systems, and coastal productivity,
then expanded eastward through administration,
education, civil engineering, and railway integration.


# ============================================================
# 2. WESTERN CORE
# ============================================================

western_core_structure:
- capital and administrative core
- education core
- railway origin core
- early industrial and civil engineering zones
- old coastal productivity belt


# ============================================================
# 3. CENTRAL BELT
# ============================================================

central_integration_belt:
- railway hubs
- industrial cities
- residential support cities
- logistics transfer points
- agricultural distribution support


# ============================================================
# 4. EASTERN STRUCTURE
# ============================================================

eastern_frontier_structure:
- underdeveloped territories
- later-stage development zones
- national defense deployment near eastern borders
- infrastructure extension corridors


# ============================================================
# 5. NATIONAL MOBILITY RULE
# ============================================================

Seiwa domestic movement is strongly supported
by dense national railway coverage.

Seiwa-made rail systems are also exported
and used widely across Civilization.


# ============================================================
# 6. FINAL RULE
# ============================================================

Seiwa is a west-origin administrative,
civil-engineering, and rail-integrated nation.
