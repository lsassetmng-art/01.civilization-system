# ============================================================
# 1. PURPOSE
# ============================================================

Defines governance posture
for Free Cities Union.


# ============================================================
# 2. CORE RULE
# ============================================================

Free Cities Union has:
- no permanent supreme ruler
- no king
- no single president-like permanent top leader

Primary governance is conducted through:
- city leader council
- inter-city deliberation
- negotiated federation decision process


# ============================================================
# 3. DECISION MODEL
# ============================================================

Recommended decision flow:
- city proposal submitted
- council discussion opened
- inter-city negotiation
- council decision adopted
- federation execution coordinated


# ============================================================
# 4. FINAL RULE
# ============================================================

Free Cities Union is a multi-city deliberative federation,
not a centralized ruler-state.
