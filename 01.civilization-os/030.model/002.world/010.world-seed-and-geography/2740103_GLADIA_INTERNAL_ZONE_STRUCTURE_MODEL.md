# ============================================================
# 1. CORE SPATIAL RULE
# ============================================================

Gladia uses a layered fortress-state spatial model
prioritizing surveillance, defense, force concentration,
and rapid response.


# ============================================================
# 2. OUTER DEFENSE BELT
# ============================================================

outer_barracks_surveillance_zone:
- dense two-story barracks belts
- border and coastal edge deployment
- surveillance posts
- pervasive security cameras
- rapid-response personnel presence


# ============================================================
# 3. SECOND DEFENSE BELT
# ============================================================

arsenal_command_defense_zone:
- arsenals
- command centers
- supply depots
- hardened defensive facilities
- anti-intrusion response centers


# ============================================================
# 4. THIRD BELT
# ============================================================

military_assembly_plaza_zone:
- military plazas
- force assembly grounds
- mobilization space
- ceremonial and control spaces


# ============================================================
# 5. FOURTH BELT
# ============================================================

controlled_residential_support_zone:
- soldier housing
- military family housing
- controlled support housing
- service residential belt


# ============================================================
# 6. CENTRAL CORE
# ============================================================

central_command_and_control_zone:
- national central command
- control facilities
- integrated surveillance center
- strategic operations center


# ============================================================
# 7. MOBILE AND AERIAL LAYERS
# ============================================================

additional_security_layers:
- mobile bases capable of moving troops and weapons
- permanent aerial drone surveillance grid


# ============================================================
# 8. MERCENARY LAYER
# ============================================================

mercenary_support_zones:
- mercenary contract zone
- contract deployment support
- para-military logistics belt
- short-stay barracks and equipment issue areas


# ============================================================
# 9. FINAL RULE
# ============================================================

Gladia is a trauma-formed fortress military state
with integrated surveillance, mobile force, and mercenary support structure.
