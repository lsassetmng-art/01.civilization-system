# ============================================================
# 1. HELIOS
# ============================================================

variation_note:
- visible hierarchy and local power pressure must remain expressible
- upper-market and worker-market retail logic may diverge strongly
- rough trust networks, local influence, and district safety can affect operations
- price and supply pressure in outer districts should feel harsher


# ============================================================
# 2. NOVA
# ============================================================

variation_note:
- sales velocity, premium branding, and campaign performance visibility should be strongest
- dashboards should emphasize revenue, margin, conversion, and customer flow
- premium retail and mass commercial retail should feel sharply separated


# ============================================================
# 3. SEIWA
# ============================================================

variation_note:
- stable household demand, orderly operations, and supplier reliability should be emphasized
- staple goods, practical value, and trust-heavy retail logic should be strong
- auditability and procedural clarity should remain visually prominent


# ============================================================
# 4. GLADIA
# ============================================================

variation_note:
- practical supply, durable goods, and readiness-oriented stocking should be emphasized
- waste tolerance should be low and discipline visibility high
- staffing and shrink control should feel tighter and more controlled


# ============================================================
# 5. ORPHEUS
# ============================================================

variation_note:
- local market trust, face-known trade, fishery-linked goods, and communal familiarity should be visible
- bargaining and local seller credibility should matter more
- small-store and island-supply logic should be supported strongly


# ============================================================
# 6. AURELIA
# ============================================================

variation_note:
- unstable supply, price stress, and survival buying should be supported
- low-trust environments, scarcity response, and damaged logistics should be expressible
- cheap staple circulation and emergency substitution should be more visible


# ============================================================
# 7. FREE CITIES UNION
# ============================================================

variation_note:
- city-by-city commercial ecology must be strongly supported
- guild retail, port retail, civic market retail, and small-shop retail may all diverge heavily
- local regulation and local reputation should shape store behavior more than federal identity


# ============================================================
# 8. FINAL RULE
# ============================================================

Retail UI must visibly change by country and city,
not only by store size or product type.
