# ============================================================
# 1. HELIOS
# ============================================================

internal_gap_notes:
- upper districts preserve larger homes, clearer room hierarchy, and symbolic guest space
- worker districts emphasize practicality and storage
- outer districts face tighter and more fragile dwelling conditions


# ============================================================
# 2. NOVA
# ============================================================

internal_gap_notes:
- executive districts live in premium private space
- ordinary dense renters live in compact, privacy-stressed units
- housing gap is one of the most visible internal differences


# ============================================================
# 3. SEIWA
# ============================================================

internal_gap_notes:
- differences exist, but orderly layout norms and family practicality reduce extreme divergence
- western core and family districts are more stable than frontier or denser zones


# ============================================================
# 4. GLADIA
# ============================================================

internal_gap_notes:
- service-linked households enjoy more stable practical layouts
- detached marginal households may still face crowded or rigid housing conditions


# ============================================================
# 5. ORPHEUS
# ============================================================

internal_gap_notes:
- island variation matters more than formal class-only difference
- richer fishing or brewing islands can support larger shared households than smaller subsistence islands


# ============================================================
# 6. AURELIA
# ============================================================

internal_gap_notes:
- protected insiders and fragile households live in sharply different spatial stability
- reconstruction pressure compresses living functions in weaker districts


# ============================================================
# 7. FREE CITIES UNION
# ============================================================

internal_gap_notes:
- port, workshop, and civic-rich cities organize domestic space very differently
- local rules and local economy shape dwelling more than federation norms


# ============================================================
# 8. FINAL RULE
# ============================================================

Internal dwelling gaps must remain visible
so that national averages do not erase lived domestic difference.
