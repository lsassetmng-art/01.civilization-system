# ============================================================
# 1. HELIOS
# ============================================================

variation_note:
- hierarchy and patronage visibility should remain strong
- worker-facing screens may feel harsher and more practical
- upper management screens may reflect favor, pressure, and informal power
- nightlife and rough patron networks may affect some civilian jobs


# ============================================================
# 2. NOVA
# ============================================================

variation_note:
- performance, speed, and profit visibility should be strongest
- dashboards should emphasize money, efficiency, and competitive ranking
- premium and elite operational views may diverge sharply from mass-worker views


# ============================================================
# 3. SEIWA
# ============================================================

variation_note:
- order, clarity, procedure, and documentation visibility should be strongest
- school, rail, public administration, and legal-process screens should feel highly structured
- auditability should be visually emphasized


# ============================================================
# 4. GLADIA
# ============================================================

variation_note:
- command clarity, readiness, and disciplined execution should dominate
- military and logistics screens should feel strict and chain-oriented
- ambiguity and decorative clutter should be minimized


# ============================================================
# 5. ORPHEUS
# ============================================================

variation_note:
- local trust, communal continuity, and relational work should remain visible
- fishery, ritual, arts, and island logistics should support softer and more local presentation
- formal hierarchy should feel weaker in many jobs


# ============================================================
# 6. AURELIA
# ============================================================

variation_note:
- instability, doctrine, and scarcity must be supportable as UI tone
- some work screens should feel constrained, fragile, or politically filtered
- protected insider roles and fragile outsider roles may need different visual logic


# ============================================================
# 7. FREE CITIES UNION
# ============================================================

variation_note:
- strong city-by-city variance must be supported
- guild, port, civic, workshop, and market-oriented work views may diverge heavily
- local identity should often override federation tone


# ============================================================
# 8. FINAL RULE
# ============================================================

Work UI must visibly change by nation and city,
not only by profession.
