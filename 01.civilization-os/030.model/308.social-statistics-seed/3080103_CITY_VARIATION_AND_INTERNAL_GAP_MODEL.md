# 1. HELIOS
# ============================================================

internal_gap_notes:
- strong gap between upper ring and outer districts
- outer districts show higher poverty and fragility
- upper households show stronger continuity and inheritance stability


# ============================================================
# 2. NOVA
# ============================================================

internal_gap_notes:
- strongest gap between high earners and dense ordinary urban workers
- premium districts and executive households differ sharply from rental-heavy urban blocks


# ============================================================
# 3. SEIWA
# ============================================================

internal_gap_notes:
- gaps exist but are institutionally moderated
- urban-rural and western-eastern development difference remains visible


# ============================================================
# 4. GLADIA
# ============================================================

internal_gap_notes:
- service-family households are more stable than marginal non-service households
- injury and service route create distinct life outcomes


# ============================================================
# 5. ORPHEUS
# ============================================================

internal_gap_notes:
- island-by-island variation is more important than class-only mapping
- kinship density changes resilience strongly


# ============================================================
# 6. AURELIA
# ============================================================

internal_gap_notes:
- collapse and reconstruction create severe local fragmentation
- stability depends on remaining state or doctrinal protection


# ============================================================
# 7. FREE CITIES UNION
# ============================================================

internal_gap_notes:
- city ecology matters more than federation-wide averages
- guild support, port access, and local rules change social outcomes sharply


# ============================================================
# 8. FINAL RULE
# ============================================================

Internal gaps must remain visible
so that national averages do not erase lived difference.
