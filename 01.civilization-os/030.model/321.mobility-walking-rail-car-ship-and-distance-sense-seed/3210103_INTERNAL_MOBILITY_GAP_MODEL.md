# ============================================================
# 1. HELIOS
# ============================================================

internal_gap_notes:
- upper circles experience distance with more privilege and lower delay cost
- worker districts rely on practical mixed transport
- outer districts experience more burden from weak access and rougher movement conditions


# ============================================================
# 2. NOVA
# ============================================================

internal_gap_notes:
- premium districts can buy faster and smoother movement
- ordinary dense districts depend on fast but crowded urban systems
- nightlife and commercial districts run on extended-hour mobility


# ============================================================
# 3. SEIWA
# ============================================================

internal_gap_notes:
- rail civilization reduces extreme mobility inequality
- frontier and less-developed eastern areas still feel farther and less smooth than core corridors


# ============================================================
# 4. GLADIA
# ============================================================

internal_gap_notes:
- service-linked households have more structured transport access
- border, arsenal, and deployment zones produce harsher mobility routines than stable civic cores


# ============================================================
# 5. ORPHEUS
# ============================================================

internal_gap_notes:
- island difference matters more than class-only difference
- ferry frequency, weather exposure, and harbor quality strongly alter everyday reachability


# ============================================================
# 6. AURELIA
# ============================================================

internal_gap_notes:
- protected zones retain more reliable movement
- fragile districts face transport unreliability, safety concerns, and weaker reach


# ============================================================
# 7. FREE CITIES UNION
# ============================================================

internal_gap_notes:
- port, guild, market, and civic-rich cities all produce different movement ecologies
- local transport culture matters more than any federal norm


# ============================================================
# 8. FINAL RULE
# ============================================================

Internal mobility gaps must remain visible
so that national averages do not erase lived reachability difference.
