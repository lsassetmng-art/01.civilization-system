# ============================================================
# 1. HELIOS
# ============================================================

naming_posture:
- Helios-prefixed hospitals and clinics
- industrial, central, western-port, research-medical tone


# ============================================================
# 2. NOVA
# ============================================================

naming_posture:
- commercial English naming
- medical center, clinical tower, executive medical tone


# ============================================================
# 3. SEIWA
# ============================================================

naming_posture:
- nation name, city name, ordinary surnames
- civic and public-trust tone

special_rule:
- Ninomiya naming is reserved for 二宮記念病院 as an honor exception


# ============================================================
# 4. GLADIA
# ============================================================

naming_posture:
- military hospital, field clinic, command medical naming


# ============================================================
# 5. ORPHEUS
# ============================================================

naming_posture:
- founder names must be avoided
- island name, harbor name, coastal and maritime naming
- each island should have at least one clinic


# ============================================================
# 6. AURELIA
# ============================================================

naming_posture:
- nation-praising or founder-praising medical naming
- doctrinal and controlled tone


# ============================================================
# 7. FREE CITIES UNION
# ============================================================

naming_posture:
- civic, city-name, market, port, mediation-oriented naming


# ============================================================
# 8. FINAL RULE
# ============================================================

Medical institution naming must align
with nation naming culture and service structure.
