# ============================================================
# 1. HELIOS
# ============================================================

naming_posture:
- Helios-prefixed names are common
- technical, industrial, research-oriented tone


# ============================================================
# 2. NOVA
# ============================================================

naming_posture:
- international commercial English naming
- academy, institute, business school, corporate training tone
- ASIC naming retained where appropriate


# ============================================================
# 3. SEIWA
# ============================================================

naming_posture:
- nation name, city name, ordinary surnames
- practical and credible civic naming
- broad public-school naming is acceptable


# ============================================================
# 4. GLADIA
# ============================================================

naming_posture:
- American-style English naming
- tactical, defense, command, training tone


# ============================================================
# 5. ORPHEUS
# ============================================================

naming_posture:
- founder names must be avoided
- island name, harbor name, maritime, ritual, coastal naming


# ============================================================
# 6. AURELIA
# ============================================================

naming_posture:
- nation-praising or founder-praising naming
- doctrinal and theocratic tone


# ============================================================
# 7. FREE CITIES UNION
# ============================================================

naming_posture:
- city-name, civic, guild, craft, ideas, mediation naming
- moderate surname usage allowed


# ============================================================
# 8. FINAL RULE
# ============================================================

Education institution naming must align
with nation naming culture and ideology.
