# ============================================================
# 1. HELIOS
# ============================================================

internal_gap_notes:
- upper households feel more status security and symbolic legitimacy
- outer households carry more grievance, insecurity, and distrust
- shared national identity exists, but lived psychology is sharply stratified


# ============================================================
# 2. NOVA
# ============================================================

internal_gap_notes:
- elite districts normalize pressure as ambition
- ordinary workers experience the same system as insecurity and comparison stress
- success and failure are psychologized personally


# ============================================================
# 3. SEIWA
# ============================================================

internal_gap_notes:
- class gaps exist, but are softened by civic structure
- urban-rural and west-east differences affect aspiration and trust tone


# ============================================================
# 4. GLADIA
# ============================================================

internal_gap_notes:
- service families internalize loyalty more strongly
- detached civilians may feel the system as control rather than belonging


# ============================================================
# 5. ORPHEUS
# ============================================================

internal_gap_notes:
- island variation matters more than formal class variation alone
- stronger kinship islands feel safer and less psychically fragmented


# ============================================================
# 6. AURELIA
# ============================================================

internal_gap_notes:
- protected ideological insiders and exposed unstable outsiders live very different emotional realities
- fear and doctrine do not distribute evenly


# ============================================================
# 7. FREE CITIES UNION
# ============================================================

internal_gap_notes:
- local civic culture shapes trust and anxiety more than federal identity
- workshop, port, and civic-rich cities produce different emotional climates


# ============================================================
# 8. FINAL RULE
# ============================================================

Internal psychology gaps must remain visible
so that national averages do not flatten lived emotional difference.
