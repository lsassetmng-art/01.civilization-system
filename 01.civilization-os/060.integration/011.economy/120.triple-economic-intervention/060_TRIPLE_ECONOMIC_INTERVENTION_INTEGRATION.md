# 1. PURPOSE

Defines integration boundaries for Triple economic intervention.

# 2. INTEGRATION SCOPE

- economic signals to intervention context
- approved adjustments to market/price/currency consumers
- reason traces to audit and review systems
- rejected intervention evidence to operations

# 3. INTEGRATION RULES

- adjustment channel must be explicit before handoff
- reason trace must accompany every approved adjustment
- undeclared intervention must not cross boundary
- ambiguous authority mode must fail closed
