# 1. PURPOSE

Defines integration boundaries for Triple world intervention.

# 2. INTEGRATION SCOPE

- intervention signals to context evaluation
- approved actions to runtime/governance channels
- reason traces to audit and operations
- rejected actions to review channels

# 3. INTEGRATION RULES

- one context binds one signal set
- published action must preserve explicit channel identity
- reason trace must accompany every approved action
- undeclared intervention must not cross boundary
