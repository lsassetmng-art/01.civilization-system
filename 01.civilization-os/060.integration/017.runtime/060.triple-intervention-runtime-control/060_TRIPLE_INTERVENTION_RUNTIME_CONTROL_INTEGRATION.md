# 1. PURPOSE

Defines integration boundaries for Triple runtime intervention.

# 2. INTEGRATION SCOPE

- Triple runtime context to signal evaluators
- approved adjustments to execution/runtime consumers
- intervention traces to audit and review systems

# 3. INTEGRATION RULES

- adjustment channel and authority must be explicit before handoff
- reason trace must accompany every approved adjustment
- undeclared intervention must not cross boundary
- ambiguous intervention state must fail closed
