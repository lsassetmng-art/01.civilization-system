# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for banking facilities,
insurance facilities,
securities facilities,
exchange facilities,
and credit-enforcement-linked finance institutions
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

Finance-related facilities must reflect:
- nation identity
- market posture
- industry posture
- trade posture
- class and wealth structure
- contract and credit culture
- urban density and enterprise concentration


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each nation and city seed should define:
- banking naming posture
- insurance naming posture
- securities naming posture
- exchange naming posture
- city-level assignment
- special financial distortions or limits if applicable


# ============================================================
# 4. FINAL RULE
# ============================================================

Finance seed must make visible
how capital, credit, risk, and investment flow
through each nation and city.
