# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for retail facility operation inside Civilization.

This architecture defines:

- retail detailed configuration
- auto operation after configuration
- intervention-based management
- inspection events
- cleanliness and hygiene management
- time-flow aligned retail simulation
- linkage to daily life and essential goods supply


# ============================================================
# 2. CORE PRINCIPLE
# ============================================================

Retail operation must not require
constant manual control.

The canonical loop is:

detailed configuration
-> auto operation
-> observation
-> intervention
-> auto operation

This architecture must remain compatible
with Earth-like real-time progression.


# ============================================================
# 3. RETAIL ROLE
# ============================================================

Retail facilities are world facilities.

They are not detached mini-games.

Retail facilities must remain connected to:

- city placement
- district demand
- household consumption
- essential goods access
- local agricultural supply
- nation policy and regulation


# ============================================================
# 4. OPERATION PRINCIPLE
# ============================================================

Auto operation must be the normal default.

Detailed management exists to:

- improve revenue
- improve margin
- improve local fit
- improve shortage resistance
- improve cleanliness and inspection outcomes
- improve local supply integration

Manual tuning must create upside,
but auto operation must remain viable.


# ============================================================
# 5. INSPECTION PRINCIPLE
# ============================================================

Retail facilities may become subject
to store inspection events.

Inspection score is 100-point based.

Pass threshold is:

90 points or higher

Inspection must evaluate at least:

- cleanliness
- hygiene
- waste control
- equipment condition
- product handling
- storage management


# ============================================================
# 6. FINAL RULE
# ============================================================

Retail management in Civilization
must be strategic configuration plus
auto-driven observation and intervention,
not constant repetitive micro-operation.
