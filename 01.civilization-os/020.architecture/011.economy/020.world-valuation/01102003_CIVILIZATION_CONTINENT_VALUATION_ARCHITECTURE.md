# ============================================================
# CIVILIZATION CONTINENT VALUATION ARCHITECTURE
# ============================================================

status: canonical
layer: 020.architecture
system: civilization-os
owner: Boss
prepared_by: Zero

purpose:
Define the continent-level valuation personality
used by CivilizationOS real-estate and geography-aware economy.

This architecture is derived from Foundation world truth.
It does not redefine world truth.
It translates world truth into valuation behavior.


# ============================================================
# 1. CORE PRINCIPLE
# ============================================================

Continent valuation must be based on Foundation truth.

Foundation defines:
- continent identity
- civilization role
- climate tendency
- economic/cultural emphasis
- state distribution

This architecture defines:
- valuation personality
- land value tendency
- rent tendency
- logistics tendency
- prestige tendency
- strategic restriction tendency

Core principle:

foundation defines what the continent is  
valuation defines how that identity affects property value


# ============================================================
# 2. CENTRAL CONTINENT VALUATION
# ============================================================

identity:
- growth
- civilization center
- dense state concentration
- political / financial / research / educational concentration

valuation personality:
- highest commercial value concentration
- highest prestige concentration
- highest urban premium concentration
- strongest finance and capital-market premium
- strong education/research district premium
- strongest mixed-use metropolitan premium

default tendencies:
- commercial_value: ultra_high
- residential_value: high
- logistics_value: high
- infrastructure_value: ultra_high
- strategic_value: medium
- scarcity_value: high
- safety_value: medium_high
- prestige_value: ultra_high

expected district patterns:
- capital districts
- financial core districts
- administrative districts
- research / education districts
- premium residential districts
- high-density mixed-use districts


# ============================================================
# 3. NORTH CONTINENT VALUATION
# ============================================================

identity:
- birth
- resources
- industry
- heavy production
- extraction and expansion energy

valuation personality:
- strongest resource-land premium
- strongest industrial corridor premium
- strongest logistics-to-port-and-factory premium
- weaker pure prestige premium than central continent
- stronger land-scale and production utility premium

default tendencies:
- commercial_value: standard
- residential_value: standard
- logistics_value: high
- infrastructure_value: high
- strategic_value: high
- scarcity_value: medium
- safety_value: medium
- prestige_value: medium
- resource_value: ultra_high
- industrial_value: ultra_high

expected district patterns:
- extraction districts
- factory belts
- industrial ports
- frontier development districts
- worker cities
- defense-support corridors


# ============================================================
# 4. SOUTH CONTINENT VALUATION
# ============================================================

identity:
- death and rebirth
- oceanic culture
- maritime life
- agriculture, fisheries, culture, memory, restoration

valuation personality:
- strongest port/oceanfront culture premium
- strong agriculture and fishery land premium
- strong tourism/culture/redevelopment premium
- mixed prestige from heritage and renewal rather than finance/power
- selective scarcity around ports, coastlines, and cultural centers

default tendencies:
- commercial_value: medium
- residential_value: medium_high
- logistics_value: medium_high
- infrastructure_value: standard
- strategic_value: medium
- scarcity_value: medium_high
- safety_value: medium
- prestige_value: medium_high
- maritime_value: ultra_high
- cultural_value: high
- redevelopment_value: high

expected district patterns:
- port cities
- fishery districts
- cultural/ritual districts
- agricultural districts
- renewal / rebirth districts
- ocean-linked trade districts


# ============================================================
# 5. CONTINENT DIFFERENTIATION RULE
# ============================================================

The continents must not be valued identically.

Central continent:
- strongest civilization-core premium

North continent:
- strongest industrial/resource premium

South continent:
- strongest maritime/cultural/redevelopment premium

This differentiation is mandatory.
Without it, world truth is economically flattened.


# ============================================================
# 6. NATION REGIME SEPARATION RULE
# ============================================================

Continent valuation is world-common.
It must not encode nation regime legality.

Examples of what must remain outside this layer:
- ownership legality
- company holding legality
- foreign holding legality
- expropriation risk
- state approval requirement

These belong to nation regime layers.


# ============================================================
# 7. REGION / DISTRICT DERIVATION RULE
# ============================================================

Region and district valuation should derive from:
- continent valuation personality
- state identity
- city role
- infrastructure concentration
- strategic restriction concentration
- local prestige/culture intensity

Continent personality is the top macro bias,
not the final local price by itself.


# ============================================================
# 8. FINAL ARCHITECTURAL RULE
# ============================================================

CivilizationOS continent valuation must reflect Foundation truth.

Core summary:

- central continent = civilization / finance / prestige premium
- north continent = resource / industry / logistics premium
- south continent = maritime / culture / renewal premium
- continent valuation is world-common
- nation regime remains separate
