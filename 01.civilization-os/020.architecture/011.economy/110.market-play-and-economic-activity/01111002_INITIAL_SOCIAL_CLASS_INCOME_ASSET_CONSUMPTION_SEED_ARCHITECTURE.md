# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for nation-level and city-level class structure,
income bands,
asset bands,
and consumption bands
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

Class, income, asset, and consumption structure must reflect:
- nation identity
- inheritance posture
- labor-market posture
- welfare posture
- housing posture
- education access
- household fragility
- business ownership concentration

These are not abstract economic labels only.
They must explain how status is lived,
how consumption differs,
and how inequality reproduces itself.


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each nation seed should define:
- class-layer structure
- income-band posture
- asset-band posture
- consumption-band posture
- mobility posture
- inheritance and business-control posture
- internal city and district gap posture


# ============================================================
# 4. FINAL RULE
# ============================================================

Social class seed must make visible
how wealth, status, ownership, and consumption
are distributed across society.
