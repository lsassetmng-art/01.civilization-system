# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for initial agriculture facilities,
fishery facilities,
food distribution facilities,
storage facilities,
and wholesale food movement
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

Agriculture, fishery, and food facilities must reflect:
- nation identity
- geography and climate
- city role
- population scale
- logistics structure
- port and rail posture
- urban food consumption density


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each nation and city seed should define:
- agriculture naming posture
- fishery naming posture
- food distribution naming posture
- storage naming posture
- city-level assignment
- cold-chain and bulk-food structure if relevant


# ============================================================
# 4. FINAL RULE
# ============================================================

Agriculture, fishery, and food seed
must make visible how people are fed,
how goods move, and how regions specialize.
