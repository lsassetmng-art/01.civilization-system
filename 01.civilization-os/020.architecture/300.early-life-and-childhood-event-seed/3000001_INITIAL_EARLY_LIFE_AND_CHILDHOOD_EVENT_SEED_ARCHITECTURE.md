# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for birth, infant care, early childhood,
childhood growth, household-dependent development,
and early education transition events
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

Early-life and childhood events must reflect:
- nation identity
- household structure
- childcare posture
- education posture
- welfare posture
- class structure
- religion and civic registration structure

These events are not generic age ticks.
They define how a new citizen enters society.


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each nation and city seed should define:
- birth event posture
- infant support event posture
- household care event posture
- nursery and kindergarten transition posture
- early aptitude or developmental evaluation posture if applicable
- family difficulty and support intervention posture
- city-level and nation-level event differences


# ============================================================
# 4. FINAL RULE
# ============================================================

Early-life seed must make visible
how society receives a newborn,
how children are supported,
and how inequality begins to shape life trajectories.
