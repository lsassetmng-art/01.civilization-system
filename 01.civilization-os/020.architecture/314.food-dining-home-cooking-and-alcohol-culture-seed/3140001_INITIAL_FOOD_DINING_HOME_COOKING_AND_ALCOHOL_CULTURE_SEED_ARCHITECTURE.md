# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for food culture,
eating-out culture,
home-cooking culture,
alcohol culture,
and everyday meal rhythm
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

Food and dining culture must reflect:
- nation identity
- geography
- labor structure
- household structure
- class structure
- religion posture
- market and nightlife posture
- agriculture and fishery base

These are not decorative tastes only.
They reveal how people live every day.


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each nation seed should define:
- staple-food posture
- home-cooking posture
- outside-meal posture
- drinking posture
- celebration-food posture
- class-linked dining difference
- city and district variation


# ============================================================
# 4. FINAL RULE
# ============================================================

Food culture seed must make visible
how work, family, class, and ritual
appear at the table.
