# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for post-graduation branching,
direct work entry,
higher-education continuation,
exam-repeat routes,
family-business succession,
state service connection,
and delayed life-course branching
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

Post-graduation branching must reflect:
- nation identity
- education posture
- hiring culture
- class structure
- family-business inheritance
- military or state-service interruption
- instability and informal labor routes

These are not simple graduation outcomes.
They define the actual conversion
from student status into adult social position.


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each nation seed should define:
- lower-secondary exit branching
- upper-secondary exit branching
- university exit branching
- exam-repeat and retry posture
- family-business succession posture
- military or state-service connection
- unstable or precarious branching posture


# ============================================================
# 4. FINAL RULE
# ============================================================

Post-graduation branching seed must make visible
how education becomes work, delay, inheritance,
service, instability, or exclusion.
