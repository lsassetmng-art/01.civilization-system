# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for factions, interest groups, lobbying channels,
patronage influence, corruption pathways,
and informal power aggregation
in Civilization seed-world planning.


# ============================================================
# 2. CORE PRINCIPLE
# ============================================================

Power aggregation must distinguish between:

- formal political organization
- policy interest groups
- patronage and protection networks
- economic influence channels
- military and doctrine pressure channels
- corruption and covert capture channels

A state may appear formally constitutional
while practical influence is factionally captured.
A state may also appear fragmented
while stable bargaining among groups
quietly structures power.


# ============================================================
# 3. REQUIRED SUBSYSTEMS
# ============================================================

The builder must include:

- Faction Type Builder
- Interest Group Builder
- Lobby Channel Builder
- Patronage Network Builder
- Corruption and Capture Builder
- Influence Weight Builder
- Conflict and Coalition Builder
- Validation Engine


# ============================================================
# 4. POWER VISIBILITY RULE
# ============================================================

The builder must separate:

- visible power groups
- semi-visible pressure groups
- hidden selection and capture groups

No nation may be modeled
as purely formal power
if informal power is socially decisive.


# ============================================================
# 5. FINAL RULE
# ============================================================

Faction design must always show
who wants what,
how they pressure the state,
and how they survive defeat.
