# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical resolution architecture
for remaining placeholder expressions
inside CivilizationOS integrated design.

This architecture resolves:

- office_specific_rule
- governance_specific_route
- draft_pending_definition placeholders
- unresolved asset reference placeholders


# ============================================================
# 2. CORE PRINCIPLE
# ============================================================

Integrated canonical documents
must not retain vague structural placeholders
when those placeholders affect:

- actor eligibility
- office legality
- power acquisition
- head of state ascent
- canonical nation profile readability


# ============================================================
# 3. RESOLUTION RULE
# ============================================================

Placeholders must be resolved
by one of the following methods:

- replace with explicit enum values
- replace with explicit rule profile values
- replace with explicit draft-state markers
- isolate unresolved asset fields
  as nation-profile asset pending fields

Free-text ambiguity
must not remain in core actor or office rules.


# ============================================================
# 4. FINAL RULE
# ============================================================

Placeholder resolution
must prioritize legality,
actor compatibility,
and power route clarity.
