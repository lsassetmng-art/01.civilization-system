# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for school progression,
exam events,
school transition events,
career branching events,
and post-compulsory education routing
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

School progression and branching must reflect:
- nation identity
- education posture
- class structure
- exam culture
- school access inequality
- labor-market connection
- household support capacity

These are not generic age transitions.
They define how children become students,
how students become candidates,
and how candidates become workers or specialists.


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each nation and city seed should define:
- primary-to-lower-secondary transition posture
- lower-secondary-to-upper-secondary transition posture
- upper-secondary branching posture
- exam intensity posture
- law / business / technical / arts branching posture
- work-entry and higher-education connection


# ============================================================
# 4. FINAL RULE
# ============================================================

School progression seed must make visible
how education becomes competition,
sorting, and life-course branching.
