# ============================================================
# 1. PURPOSE
# ============================================================

This document defines how Foundation-level symbolic meaning
is translated into Civilization world-seed planning.

It does not replace Foundation.
It translates Foundation into usable initial world design signals.


# ============================================================
# 2. CORE RULE
# ============================================================

Foundation meaning must influence:
- continent identity
- initial nation posture
- city-role distribution
- industry bias
- education bias
- cultural and ritual density
- system service concentration

This influence is an initial-value guide,
not an unchangeable runtime destiny.


# ============================================================
# 3. CONTINENT TRANSLATION
# ============================================================
