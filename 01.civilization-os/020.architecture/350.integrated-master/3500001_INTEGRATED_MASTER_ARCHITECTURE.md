# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical integrated master layer
for Civilization seed-world design.

This layer consolidates:

- nation identity and variation
- event modifier references
- UI slot references
- job and work references
- quick-reference nation summaries
- cross-nation comparison matrix

into a unified read model
for design, balancing, UI, and future implementation.


# ============================================================
# 2. CORE PRINCIPLE
# ============================================================

Integrated masters must not replace
detailed design documents.

Integrated masters must:

- summarize
- map
- normalize
- cross-reference
- expose lookup-ready structure

Detailed design remains the source layer.
Integrated master remains the reference layer.


# ============================================================
# 3. REQUIRED SUBSYSTEMS
# ============================================================

The integrated master layer must include:

- Nation Master
- Event Modifier Master
- UI Slot Master
- Job Master
- Country Quick Reference
- Civilization Society Matrix


# ============================================================
# 4. FINAL RULE
# ============================================================

Integrated master documents must make it possible
to understand the whole civilization quickly
without erasing nation-specific depth.
