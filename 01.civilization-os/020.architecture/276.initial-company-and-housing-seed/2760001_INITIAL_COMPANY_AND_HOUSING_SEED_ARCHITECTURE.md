# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for initial company placement and housing placement
in the Civilization seed world.


# ============================================================
# 2. CORE RULE
# ============================================================

Initial placement must reflect:
- nation character
- city role
- population scale
- zoning posture
- industry structure
- historical and political traces

This layer does not require full micro-address placement.
It defines structured initial distribution.


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each seed placement profile should define:
- anchor company types
- system service access posture
- housing type balance
- ownership and rental tendency
- mixed-use tendency
- redevelopment tendency
- public facility adjacency tendency


# ============================================================
# 4. FINAL RULE
# ============================================================

Initial company and housing seed must be city-aware,
nation-aware, and population-aware.
