# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for media consumption,
information trust,
rumor spread,
state messaging,
religious messaging,
and everyday information contact culture
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

Media and information culture must reflect:
- nation identity
- civic trust
- religion posture
- education posture
- political openness
- urban density
- class structure
- rumor and social network dependence


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each nation seed should define:
- official media trust posture
- rumor reliance posture
- local word-of-mouth posture
- political messaging posture
- commercial media posture
- class and district information gap posture


# ============================================================
# 4. FINAL RULE
# ============================================================

Media seed must make visible
how truth, rumor, authority, and belonging
move through society.
