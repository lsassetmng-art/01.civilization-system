# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for adult life-course timing,
marriage timing,
childbirth timing,
housing acquisition timing,
and household stabilization timing
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

Life-course timing must reflect:
- nation identity
- education duration
- hiring culture
- military interruption
- housing access
- family norms
- class and inheritance structure
- non-regular work prevalence


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each nation seed should define:
- first job timing posture
- marriage timing posture
- childbirth timing posture
- rental-to-ownership timing posture
- family-business inheritance timing posture
- delayed adulthood posture
- unstable household formation posture


# ============================================================
# 4. FINAL RULE
# ============================================================

Life-course seed must make visible
how citizens become workers,
partners, parents, and household builders.
