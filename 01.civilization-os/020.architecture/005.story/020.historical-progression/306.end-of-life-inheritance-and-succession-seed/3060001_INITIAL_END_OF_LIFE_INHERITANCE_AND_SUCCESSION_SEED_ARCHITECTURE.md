# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for death events,
funeral routing,
inheritance events,
family-business succession,
estate distribution,
and post-death household transition
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

End-of-life, inheritance, and succession must reflect:
- nation identity
- religion and memorial posture
- household structure
- property regime
- class structure
- family-business logic
- legal and civic registration structure

These are not simple end states.
They define how the dead are processed,
how households survive loss,
and how power and assets continue.


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each nation seed should define:
- death registration posture
- funeral routing posture
- inheritance posture
- family-business succession posture
- fragile-household transition posture
- state intervention posture if applicable


# ============================================================
# 4. FINAL RULE
# ============================================================

End-of-life seed must make visible
how death changes households, property,
status, and institutional continuity.
