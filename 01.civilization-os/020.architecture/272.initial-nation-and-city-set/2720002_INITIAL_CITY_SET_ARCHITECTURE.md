# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for the initial city set of major nations.


# ============================================================
# 2. CORE RULE
# ============================================================

Initial cities must be explicit seed-world units.

Major nation city sets must include:
- core cities already fixed by Foundation truth
- expanded support cities for population, housing,
  industry, ports, culture, and infrastructure


# ============================================================
# 3. SPECIAL EXPANSION RULES
# ============================================================

Aurelia city naming:
- eastern / russian-like naming posture
- strong religious-historical tone

Free Cities Union:
- 12-city structure
- no permanent supreme ruler
- each city has its own leader
- federation decisions are made by council deliberation


# ============================================================
# 4. FINAL RULE
# ============================================================

Initial city set must support population scale,
housing realism, industry distribution,
and political identity.
