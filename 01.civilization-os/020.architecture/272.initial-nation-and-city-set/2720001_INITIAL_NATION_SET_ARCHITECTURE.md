# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for the initial nation set of Civilization.

This layer imports Foundation truth
into explicit seed-world nation planning.


# ============================================================
# 2. CORE RULE
# ============================================================

Initial nations are not only builder possibilities.

They are explicit seed-world canonical entities
with fixed initial identity, placement,
major cities, population posture,
industry posture, and governance posture.


# ============================================================
# 3. INCLUDED MAJOR NATIONS
# ============================================================

The initial major nation set includes:

- helios_democratic_kingdom
- nova_commercial_federation
- seiwa_state
- gladia_military_alliance
- orpheus_oceanic_union
- aurelia_federal_republic
- free_cities_union


# ============================================================
# 4. SPECIAL CASES
# ============================================================

aurelia_federal_republic:
- historical / dismantled state in current runtime posture

free_cities_union:
- 12-city union
- no permanent supreme ruler
- governed by inter-city representative council


# ============================================================
# 5. FINAL RULE
# ============================================================

Initial nation set is a canonical seed-world layer
based on Foundation truth and world-seed design.
