# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for initial population structure
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

Population must not be treated
as total headcount only.

Initial population structure must reflect:
- age bands
- household composition
- labor participation
- student population
- elderly population
- income distribution
- education distribution


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each nation and city population seed should define:
- age band posture
- household posture
- labor posture
- income posture
- education posture
- special demographic distortions if applicable


# ============================================================
# 4. FINAL RULE
# ============================================================

Population structure is a first-class world-seed layer
that drives housing, schools, labor, and welfare.
