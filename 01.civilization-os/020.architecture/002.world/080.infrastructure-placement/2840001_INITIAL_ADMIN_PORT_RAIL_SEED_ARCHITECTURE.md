# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for initial administrative facilities,
port facilities, and railway facilities
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

Administrative, port, and railway facilities must reflect:
- nation identity
- capital structure
- city role
- logistics posture
- political and military control posture
- historical traces

These facilities are core structural institutions,
not optional decorative objects.


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each seed should define:
- central administrative facility names
- city administrative facility names
- port authority and harbor naming
- railway station, hub, and control naming
- city-level assignment of these facilities


# ============================================================
# 4. FINAL RULE
# ============================================================

Administrative, port, and railway seed
must explain how the civilization is actually governed and moved.
