# ============================================================
# 1. PURPOSE
# ============================================================

Defines the official architecture for Exhibition Builder.

This builder supports digital art exhibitions,
screenings, gallery events, decoration,
equipment placement, and artwork layout.


# ============================================================
# 2. CORE UI SECTIONS
# ============================================================

Supported sections:
- venue selection
- room / zone selection
- decoration selection
- equipment placement
- artwork placement
- route design
- lighting design
- capacity and safety validation
- opening schedule


# ============================================================
# 3. FINAL RULE
# ============================================================

Exhibition Builder is a first-class spatial editing UI,
not only an event metadata editor.
