# ============================================================
# 1. PURPOSE
# ============================================================

Define the canonical architecture
for cross-layer propagation
inside Civilization.


# ============================================================
# 2. CORE PRINCIPLE
# ============================================================

Civilization systems
must support propagation across layers.

Representative propagation families include:

- disaster propagation
- epidemic propagation
- migration propagation
- infrastructure-failure propagation
- economic shock propagation
- crime and instability propagation
- logistics disruption propagation


# ============================================================
# 3. LAYER RELATION
# ============================================================

Propagation may occur across:

- space and planetary layers
- continent and state layers
- state and city layers
- city and district layers
- infrastructure and facility layers
- facility and actor layers


# ============================================================
# 4. FINAL RULE
# ============================================================

Multi-layer propagation architecture must remain:

- cross-layer
- event-capable
- failure-capable
- epidemic-capable
- shock-capable
