# ============================================================
# 1. PURPOSE

Define the canonical architecture
for flora and fauna progression
inside Civilization.


# ============================================================
# 2. CORE PRINCIPLE
# ============================================================

Plants and animals
must be treated as active life-system layers.

Progression may include:

- growth
- reproduction
- illness
- decline
- death
- habitat movement
- domesticated productivity change
- ecosystem stress response


# ============================================================
# 3. INPUTS
# ============================================================

Representative inputs include:

- season
- climate
- water
- food availability
- habitat quality
- disaster effect
- disease effect
- human and city pressure
- agricultural control where applicable


# ============================================================
# 4. OUTPUTS
# ============================================================

Representative outputs include:

- crop yield posture
- livestock condition posture
- habitat change posture
- food-chain stress posture
- agricultural impact
- ecosystem balance posture


# ============================================================
# 5. FINAL RULE
# ============================================================

Flora and fauna progression architecture must remain:

- life-cycle-aware
- habitat-aware
- agriculturally relevant
- ecosystem-aware
