# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for crime events,
accident events,
illness events,
job-loss events,
bankruptcy events,
and household disruption events
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

Risk and disruption events must reflect:
- nation identity
- welfare posture
- labor structure
- legal structure
- medical support structure
- family resilience
- housing and debt fragility
- public safety and instability level


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each nation seed should define:
- crime exposure posture
- accident exposure posture
- illness and care disruption posture
- unemployment and job-loss posture
- debt and bankruptcy posture
- fragile household recovery posture


# ============================================================
# 4. FINAL RULE
# ============================================================

Risk seed must make visible
how ordinary life is damaged,
how households break,
and how societies absorb or fail to absorb crisis.
