# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for initial tourism facilities,
entertainment facilities,
and cultural facilities
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

Tourism, entertainment, and culture facilities must reflect:
- nation identity
- city role
- religion and ritual posture
- nightlife and consumption posture
- memorial and historical posture
- education and art posture

These facilities are structural city-identity institutions,
not optional decoration.


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each nation and city seed should define:
- tourism naming posture
- entertainment naming posture
- cultural naming posture
- city-level assignment
- special ritual or nightlife distinctions if applicable


# ============================================================
# 4. FINAL RULE
# ============================================================

Tourism, entertainment, and culture seed
must make leisure, memory, art, and civic expression visible.
