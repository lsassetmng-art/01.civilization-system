# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for relationship support facilities,
marriage support facilities,
wedding facilities,
newlywed support facilities,
and early household formation support
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

Marriage and family-formation facilities must reflect:
- nation identity
- household structure
- religion and ritual posture
- class structure
- housing access
- employment and life-course timing
- urban and island accessibility


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each nation and city seed should define:
- relationship and matchmaking naming posture
- marriage support naming posture
- wedding facility naming posture
- newlywed housing support naming posture
- city-level assignment


# ============================================================
# 4. FINAL RULE
# ============================================================

Marriage and family-formation seed must make visible
how relationships are recognized,
how households are formed,
and how new family life becomes stable.
