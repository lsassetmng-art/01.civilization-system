# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for eldercare facilities,
long-term care facilities,
retirement support structures,
and end-of-life support facilities
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

Eldercare and end-of-life support must reflect:
- nation identity
- household structure
- family responsibility norms
- civic welfare posture
- labor and retirement logic
- religion and memorial culture
- urban and island accessibility

These facilities are not generic welfare objects.
They reveal how each society understands aging,
dependency, dignity, and final support.


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each nation and city seed should define:
- eldercare naming posture
- family-care vs facility-care posture
- end-of-life support posture
- retirement support posture
- city-level assignment
- national exceptions if applicable


# ============================================================
# 4. FINAL RULE
# ============================================================

Eldercare and end-of-life seed must make visible
how old age is lived,
how dependency is supported,
and how final life stages are institutionally handled.
