# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for initial childcare facilities,
early childhood facilities,
family support facilities,
and infant-parent support facilities
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

Childcare and early-family support facilities must reflect:
- nation identity
- age structure
- household structure
- work style
- education posture
- religion and civic welfare posture
- transport and neighborhood accessibility


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each nation and city seed should define:
- nursery naming posture
- kindergarten naming posture
- childcare support naming posture
- infant-parent support naming posture
- city-level assignment
- special institutional distortions if applicable


# ============================================================
# 4. FINAL RULE
# ============================================================

Childcare seed must make visible
how early life, family support, and working households
are sustained in each nation.
