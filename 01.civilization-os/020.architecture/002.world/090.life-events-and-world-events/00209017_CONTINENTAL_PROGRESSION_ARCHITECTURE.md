# ============================================================
# 1. PURPOSE
# ============================================================

Define the canonical architecture
for autonomous continental progression
inside Civilization.


# ============================================================
# 2. CORE PRINCIPLE
# ============================================================

Continents must be treated as
active macro-geographic progression units.

Continental progression may include:

- climate-zone tendency
- population drift
- trade-zone structure
- conflict-zone structure
- epidemic spread posture
- resource concentration posture
- cultural and civilizational tendency


# ============================================================
# 3. INPUTS
# ============================================================

Representative continental inputs include:

- planetary environment
- inter-state trade
- migration flow
- war posture
- epidemic posture
- resource distribution
- infrastructure corridors
- city concentration
- continent-scale policy or alliance posture


# ============================================================
# 4. OUTPUTS
# ============================================================

Representative continental outputs include:

- continental growth/decline posture
- region imbalance posture
- trade concentration posture
- broad instability posture
- large-scale migration pressure
- broad environmental risk concentration


# ============================================================
# 5. FINAL RULE
# ============================================================

Continental progression architecture must remain:

- macro-geographic
- multi-state-aware
- migration-aware
- trade-aware
- instability-aware
