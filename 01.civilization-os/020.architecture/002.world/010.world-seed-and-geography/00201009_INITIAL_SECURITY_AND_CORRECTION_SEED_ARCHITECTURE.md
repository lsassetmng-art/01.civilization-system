# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for initial military facilities,
security facilities,
and correction facilities
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

Security and correction facilities must reflect:
- nation identity
- military posture
- border and maritime exposure
- ideological posture
- public order structure
- correction philosophy
- historical trauma and conflict posture

These facilities are structural institutions,
not optional edge objects.


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each nation and city seed should define:
- major military facility names
- security / policing facility names
- correction facility names
- city-level assignment
- historical vs current runtime distinction if needed


# ============================================================
# 4. FINAL RULE
# ============================================================

Security and correction seed must make coercive power,
public order, and state control visible in the world model.
