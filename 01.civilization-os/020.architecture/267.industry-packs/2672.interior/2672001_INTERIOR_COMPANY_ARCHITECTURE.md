# ============================================================
# 1. PURPOSE
# ============================================================

Defines the official architecture for interior_company.

This company type creates interior and housing exterior related
design output for spaces, units, and lived environments.


# ============================================================
# 2. CORE RESPONSIBILITIES
# ============================================================

- space brief intake
- interior concept creation
- room/unit design planning
- fit-out guidance
- revision handling
- delivery of approved design package


# ============================================================
# 3. OUTPUT POSTURE
# ============================================================

Primary outputs:
- housing_interior_design
- housing_exterior_design


# ============================================================
# 4. SITE POSTURE
# ============================================================

Required:
- headquarters

Recommended:
- office
- studio


# ============================================================
# 5. FINAL RULE
# ============================================================

interior_company is a design company type.
It may specify fit-out direction,
but does not execute construction directly.
