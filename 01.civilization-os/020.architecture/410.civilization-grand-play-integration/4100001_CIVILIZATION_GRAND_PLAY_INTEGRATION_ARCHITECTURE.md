# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical integration architecture
for the six major play domains of Civilization.


# ============================================================
# 2. PLAY DOMAINS
# ============================================================

play_domains:
- political_power_and_election
- economic_and_market_play
- social_order_and_ideology_play
- military_and_diplomacy_play
- media_and_information_war_play
- personal_rise_and_fall_play


# ============================================================
# 3. INTEGRATION PRINCIPLE
# ============================================================

These play domains must not remain isolated.

Political change affects market and war.
War affects social order and media.
Media affects elections and market trust.
Personal ascent unlocks state power.
State power reshapes all other domains.


# ============================================================
# 4. FINAL RULE
# ============================================================

Civilization grand play
must behave like one connected world,
not six disconnected feature sets.
