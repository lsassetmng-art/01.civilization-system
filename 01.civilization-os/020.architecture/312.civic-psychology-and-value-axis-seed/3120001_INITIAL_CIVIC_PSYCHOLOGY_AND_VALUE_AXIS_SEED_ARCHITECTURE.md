# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for civic psychology,
value-axis posture,
happiness tendency,
loyalty tendency,
faith tendency,
competition tendency,
and social-emotional orientation
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

Civic psychology and value axes must reflect:
- nation identity
- religion posture
- class structure
- labor and competition culture
- welfare posture
- military or civic duty posture
- family and kinship structure
- insecurity and instability level

These are not abstract personality labels.
They must explain how societies feel,
what they reward,
and what kind of inner pressure citizens live under.


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each nation seed should define:
- happiness tendency
- anxiety tendency
- civic trust tendency
- loyalty tendency
- faith tendency
- competition tendency
- family-dependence tendency
- social conformity tendency
- self-direction tendency
- grievance or resentment sensitivity if relevant


# ============================================================
# 4. FINAL RULE
# ============================================================

Civic psychology seed must make visible
how institutions become inner values,
stress, trust, and life meaning.
