# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for daily time rhythm,
morning-night preference,
punctuality culture,
appointment culture,
lateness tolerance,
and work-life boundary culture
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

Time-rhythm culture must reflect:
- nation identity
- labor posture
- nightlife posture
- school posture
- military posture
- family culture
- city density
- island and transport conditions

These are not scheduling notes only.
They reveal discipline, pressure, flexibility,
fatigue, and the social meaning of time.


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each nation seed should define:
- morningness posture
- lateness tolerance posture
- appointment precision posture
- work-life boundary posture
- after-hours obligation posture
- family-time protection posture
- city and class time-gap posture


# ============================================================
# 4. FINAL RULE
# ============================================================

Time-rhythm culture seed must make visible
how societies organize obligation, delay, rest, and daily tempo.
