# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for clothing culture,
uniform culture,
class-linked dress,
formalwear culture,
and visible appearance norms
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

Appearance and clothing culture must reflect:
- nation identity
- climate and geography
- labor structure
- class structure
- school and company culture
- military influence
- religion and ritual posture
- urban versus island life

These are not decorative fashion notes only.
They reveal hierarchy, duty, belonging, and everyday habit.


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each nation seed should define:
- everyday clothing posture
- workwear posture
- school uniform posture
- company attire posture
- formalwear posture
- ritualwear posture
- class and district appearance gap posture


# ============================================================
# 4. FINAL RULE
# ============================================================

Appearance culture seed must make visible
how hierarchy, profession, ritual, and ordinary life
appear on the body.
