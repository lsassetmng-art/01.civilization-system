# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for living-space culture,
room layout culture,
house-size expectations,
cohabitation norms,
and household interior use patterns
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

Living-space and cohabitation culture must reflect:
- nation identity
- class structure
- housing posture
- family culture
- eldercare posture
- childcare posture
- climate and geography
- urban density and island life

These are not just floor plans.
They reveal privacy, hierarchy, burden, and everyday family life.


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each nation seed should define:
- house-size expectation posture
- room separation posture
- multi-generational cohabitation posture
- privacy posture
- household work-sharing posture
- child room and elder room posture
- class-linked dwelling gap posture


# ============================================================
# 4. FINAL RULE
# ============================================================

Living-space culture seed must make visible
how households physically live together,
separate, endure, and reproduce daily life.
