# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for beauty standards,
cleanliness norms,
scent culture,
makeup culture,
hair culture,
and visible body presentation
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

Body-aesthetic culture must reflect:
- nation identity
- class structure
- labor posture
- religion posture
- nightlife posture
- military posture
- school and workplace norms
- climate and living conditions

These are not decorative notes only.
They reveal discipline, attraction, status,
purity, labor, and self-presentation.


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each nation seed should define:
- cleanliness expectation posture
- scent visibility posture
- makeup visibility posture
- hair-style control posture
- body-modification or ornament posture
- class-linked beauty gap posture
- public versus private grooming difference posture


# ============================================================
# 4. FINAL RULE
# ============================================================

Body-aesthetic culture seed must make visible
how societies judge bodies,
care, attractiveness, and presentation.
