# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for everyday mobility culture,
walking culture,
rail culture,
car culture,
ship culture,
and distance perception
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

Mobility culture must reflect:
- nation identity
- geography
- urban density
- transport infrastructure
- class structure
- labor rhythm
- island versus continental life
- military and industrial structure

These are not only transport notes.
They reveal how people measure effort,
time, distance, and reachable life.


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each nation seed should define:
- walking baseline posture
- rail reliance posture
- road and car reliance posture
- ship and ferry reliance posture
- long-distance travel posture
- commute tolerance posture
- class-linked mobility gap posture


# ============================================================
# 4. FINAL RULE
# ============================================================

Mobility culture seed must make visible
how people move through ordinary life,
how far "near" feels,
and what infrastructure shapes daily possibility.
