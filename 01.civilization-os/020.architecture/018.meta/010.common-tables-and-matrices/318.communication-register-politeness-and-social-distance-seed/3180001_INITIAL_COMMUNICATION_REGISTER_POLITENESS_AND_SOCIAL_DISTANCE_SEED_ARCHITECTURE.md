# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for speech culture,
politeness culture,
social distance in conversation,
public-private register switching,
and everyday communication norms
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

Communication culture must reflect:
- nation identity
- class structure
- school and workplace culture
- military or civic discipline
- family culture
- religion posture
- nightlife and friendship culture
- hierarchy and power visibility

These are not only linguistic notes.
They reveal how people measure respect,
distance, familiarity, authority, and belonging.


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each nation seed should define:
- politeness posture
- hierarchy-sensitive speech posture
- public-private register gap
- directness posture
- emotional expression posture
- friendliness and familiarity posture
- class-linked speech difference posture


# ============================================================
# 4. FINAL RULE
# ============================================================

Communication culture seed must make visible
how hierarchy, friendship, duty, and distance
are expressed in speech.
