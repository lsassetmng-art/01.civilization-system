# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for leisure culture,
holiday behavior,
friendship culture,
play culture,
and ordinary non-work daily-life behavior
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

Leisure and friendship culture must reflect:
- nation identity
- labor rhythm
- food and drinking culture
- family culture
- city density
- religion and festival posture
- class structure
- island versus continental life

These are not optional flavor notes.
They explain how people rest, bond, escape pressure,
and build social identity outside formal duty.


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each nation seed should define:
- holiday-use posture
- after-work posture
- friendship density posture
- family-outing posture
- drinking-and-nightlife posture
- arts and play posture
- class-linked leisure difference posture


# ============================================================
# 4. FINAL RULE
# ============================================================

Leisure culture seed must make visible
how societies rest, play, gather, and emotionally recover.
