# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for religious observances,
national holidays,
seasonal civic festivals,
memorial days,
and calendar-linked public events
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

Calendar events must reflect:
- foundation theology
- nation identity
- public memory
- labor and school rhythm
- mourning and celebration culture
- urban and island civic life
- market and tourism effects

Calendar structure is not decorative.
It is a runtime social system.


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each nation and civilization-wide calendar seed should define:
- major religious observances
- national holidays
- civic and seasonal festivals
- memorial days
- calendar effects on schools, labor, public facilities, and markets


# ============================================================
# 4. FINAL RULE
# ============================================================

Religious and public calendar seed must make visible
how time itself is socially structured.
