# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for initial sports facilities,
exercise facilities,
life-leisure facilities,
and civic recreation facilities
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

Sports and life-leisure facilities must reflect:
- nation identity
- city role
- age structure
- household structure
- training posture
- class and density structure
- school and work culture


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each nation and city seed should define:
- sports naming posture
- exercise facility naming posture
- civic recreation naming posture
- city-level assignment
- distinction between military training and civilian sport if relevant


# ============================================================
# 4. FINAL RULE
# ============================================================

Sports and leisure seed must make visible
how bodies are trained, maintained, entertained,
and socially organized.
