# ============================================================
# 1. PURPOSE
# ============================================================

Defines the official architecture for culture_arts_company.

culture_arts_company is a user-creatable company type
that creates story and art output
and may operate exhibitions and screenings.


# ============================================================
# 2. CORE RESPONSIBILITIES
# ============================================================

- produce creative works
- manage portfolio and release lifecycle
- operate exhibitions and screenings
- manage rights and canon-linked relations
- support scout and official adoption flows


# ============================================================
# 3. SITE POSTURE
# ============================================================

Required:
- headquarters
- studio or gallery_site

Optional:
- branch
- gallery_site
- service_site


# ============================================================
# 4. OUTPUT POSTURE
# ============================================================

Primary outputs:
- novel_content
- manga_content
- anime_content
- drama_content
- film_content
- digital_art_content
- exhibition_event
- screening_event


# ============================================================
# 5. FINAL RULE
# ============================================================

culture_arts_company combines creative production
and public cultural presentation.
