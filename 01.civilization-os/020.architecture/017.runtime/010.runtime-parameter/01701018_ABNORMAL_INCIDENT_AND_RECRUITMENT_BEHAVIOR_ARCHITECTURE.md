# ============================================================
# 1. PURPOSE
# ============================================================

Define the canonical architecture
for abnormal behavior,
incident involvement,
and recruitment or scouting events
inside Civilization.


# ============================================================
# 2. CORE PRINCIPLE
# ============================================================

Autonomous behavior
must include not only ordinary routine behavior
but also exceptional and disruptive behavior.

This includes:

- mental abnormality posture
- incident causation posture
- incident victim posture
- witness / report posture
- recruitment or scouting posture


# ============================================================
# 3. ABNORMAL BEHAVIOR FAMILIES
# ============================================================

Representative abnormal behavior families include:

- mental instability
- schedule collapse
- socially disruptive behavior
- incident-causing behavior
- self-harm-risk-adjacent crisis posture where permitted by world rules
- extreme stress deviation


# ============================================================
# 4. INCIDENT INVOLVEMENT FAMILIES
# ============================================================

Representative incident involvement families include:

- causing an incident
- being victimized
- being drawn into an incident
- witnessing an incident
- reporting an incident
- escape or protective action


# ============================================================
# 5. RECRUITMENT AND SCOUTING
# ============================================================

Representative recruitment or scouting posture includes:

- receiving a scout approach
- receiving a work or role offer
- being invited into an organization
- being politically recruited
- changing affiliation due to recruitment

Recruitment must remain
actor-specific and context-sensitive.


# ============================================================
# 6. FINAL RULE
# ============================================================

Abnormal, incident, and recruitment behavior architecture must remain:

- non-routine-capable
- incident-aware
- recruitment-aware
- actor-specific
- connected to personality, environment, and history
