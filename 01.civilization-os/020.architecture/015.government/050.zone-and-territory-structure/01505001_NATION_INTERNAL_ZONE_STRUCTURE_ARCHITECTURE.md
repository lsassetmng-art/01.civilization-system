# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for internal zone structure of major nations.

This layer describes:
- capital-core structure
- inner urban rings
- military and industrial placement
- housing distribution logic
- mixed-use posture
- historical spatial traces


# ============================================================
# 2. CORE RULE
# ============================================================

Nation internal zone structure must not be treated
as generic zoning only.

It must preserve:
- political character
- historical traces
- security logic
- housing hierarchy
- industrial and market geography
- symbolic and ritual geography


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each major nation internal-zone profile should define:
- capital_core_structure
- outer_ring_structure
- military_spatial_structure
- industrial_spatial_structure
- housing_spatial_structure
- commercial_spatial_structure
- historical_trace_structure
- zoning_uniformity_or_variability
- special_state_security_layers


# ============================================================
# 4. FINAL RULE
# ============================================================

Nation internal spatial structure is a first-class
world-seed design layer and must be nation-specific.
