# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical extension architecture
for Power Structure Builder
inside Nation Builder.

This extension separates:

- constitutional structure definition
- mutable power-scope operation
- regime-change event control

so that election-based states do not allow
ordinary post-creation rewriting
of their constitutional foundation.


# ============================================================
# 2. CORE PRINCIPLE
# ============================================================

Nation Builder must distinguish between:

1. structure definition
2. scope operation
3. regime-change transition

These three layers must never be merged.


# ============================================================
# 3. THREE-LAYER MODEL
# ============================================================

The Power Structure Builder must consist of:

- Structure Locked Layer
- Scope Editable Layer
- Regime Change Event Layer

Their meanings are:

Structure Locked Layer
- constitutional and foundational state design
- defined at nation creation
- locked during normal operation

Scope Editable Layer
- legal operational range of power
- delegation
- emergency activation state
- ministry and authority distribution
- editable after nation creation within allowed rules

Regime Change Event Layer
- exceptional transition layer
- used only by special events
- constitutional reform
- revolution
- coup
- emergency reconstruction
- restoration
- regime collapse


# ============================================================
# 4. LOCK RULE
# ============================================================

For election-based nations,
post-creation editing must not directly modify:

- regime type
- election basis
- legislature existence
- executive selection basis
- judiciary base structure
- constitutional term structure
- amendment base rule

These are structure-layer elements.

Normal post-creation editing may only modify:

- authority scope
- delegation range
- active emergency power state
- ministry-level operational range
- local discretion range
- institutional influence weights
- practical control intensity

if such modification remains legal
within the state's structure definition.


# ============================================================
# 5. UI MODE RULE
# ============================================================

The builder must expose three modes:

A. Constitution Mode
- available at nation creation
- available only through exceptional reform events later
- edits locked structure

B. Governance Operation Mode
- normal post-creation mode
- edits scope only
- cannot alter constitutional foundations

C. Regime Change Mode
- event-only mode
- triggered by explicit reform or collapse conditions
- may unlock structure transition workflows


# ============================================================
# 6. REQUIRED SUBSYSTEMS
# ============================================================

The extension must include:

- State Structure Builder
- Legislature Builder
- Executive Builder
- Judiciary Builder
- Local Governance Builder
- Elite and Influence Builder
- Emergency Rule Builder
- Appointment and Succession Builder
- Validation and Lock Engine


# ============================================================
# 7. COUNTRY FIT
# ============================================================

This model must support at least the following cases:

Helios
- formal monarchy + legislature
- king influence is partly informal
- old noble dominance remains visible

Nova
- formal commercial-federal structure
- finance and ASIC influence must be representable
- structure mostly locked after creation

Seiwa
- bureaucratic legal-administrative state
- procedural structure strongly locked

Gladia
- command-heavy state
- emergency power range is large
- military command influence remains highly editable in scope layer

Orpheus
- weak central union
- island autonomy weights are scope-editable
- core union structure remains locked

Aurelia
- reconstruction or collapse mode may temporarily widen regime-change access

Free Cities Union
- federation-level structure locked
- city autonomy scope strongly editable


# ============================================================
# 8. FINAL RULE
# ============================================================

The Power Structure Builder must treat
state form, power operation, and regime transition
as separate canonical layers.
