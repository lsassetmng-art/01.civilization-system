# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for Legislature Builder
inside Nation Builder and Power Structure Builder.

This builder defines:

- chamber structure
- seat structure
- member selection basis
- legislative authority range
- budget authority
- impeachment authority
- committee structure
- dissolution and term rules


# ============================================================
# 2. CORE PRINCIPLE
# ============================================================

Legislature Builder must separate:

- constitutional chamber design
- seat allocation logic
- member entry logic
- legislative power scope
- practical political influence

A chamber may exist formally
while real influence is weak.
A chamber may also be structurally weak
but practically dominant.


# ============================================================
# 3. REQUIRED SUBSYSTEMS
# ============================================================

The builder must include:

- Chamber Structure Builder
- Seat Composition Builder
- Member Entry Builder
- Legislative Power Builder
- Committee Builder
- Dissolution and Term Builder
- Legislature Influence Builder
- Legislature Validation Engine


# ============================================================
# 4. LAYER OWNERSHIP
# ============================================================

Structure-layer examples:
- unicameral or bicameral
- upper house existence
- lower house existence
- chamber naming
- seat classes
- core legislative powers
- dissolution rule base
- term base

Scope-layer examples:
- current legislature influence strength
- practical veto friction
- committee activity level
- elite capture level
- party discipline strength
- budget negotiation intensity

Transition-layer examples:
- upper house abolition
- emergency suspension
- chamber redesign
- revolutionary reconstitution


# ============================================================
# 5. FINAL RULE
# ============================================================

The Legislature Builder must support
formal democracy,
elite-dominated chamber systems,
federal councils,
city unions,
command-shadow legislatures,
and ceremonial legislatures
without confusing form and reality.
