# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for political participation,
civic voice,
public opinion formation,
obedience and dissent,
and state-society emotional response
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

Political participation and public opinion must reflect:
- nation identity
- civic trust
- loyalty posture
- fear posture
- media posture
- class structure
- repression or openness
- local versus national identity


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each nation seed should define:
- election participation posture
- petition and complaint posture
- protest posture
- state loyalty posture
- fear-of-state posture
- public opinion volatility posture
- class-linked political voice gap posture


# ============================================================
# 4. FINAL RULE
# ============================================================

Political seed must make visible
how citizens obey, negotiate, resent,
or try to shape power.
