# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for disability support facilities,
rehabilitation facilities,
daily-life support facilities,
and disability-linked work support facilities
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

Disability, rehabilitation, and work-support facilities
must align with:
- eldercare posture
- nation identity
- household structure
- labor structure
- medical posture
- civic welfare posture
- family vs institutional support logic


# ============================================================
# 3. FINAL RULE
# ============================================================

Disability support seed must make visible
how each society supports function, dignity,
daily life, and work participation.
