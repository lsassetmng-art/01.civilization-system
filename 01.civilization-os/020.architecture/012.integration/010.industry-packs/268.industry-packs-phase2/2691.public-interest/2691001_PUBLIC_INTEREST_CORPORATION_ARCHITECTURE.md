# ============================================================
# 1. PURPOSE
# ============================================================

Defines the official architecture for public_interest_corporation.

This system-limited company type exists for public mission execution,
non-standard public service delivery, and public-purpose institutional activity.


# ============================================================
# 2. CORE RULE
# ============================================================

public_interest_corporation is system company only
and base non-listable.


# ============================================================
# 3. CORE RESPONSIBILITIES
# ============================================================

- public mission execution
- subsidy or donation linked finance
- public-service-aligned operation
- non-listable institutional governance


# ============================================================
# 4. FINAL RULE
# ============================================================

Public-interest corporation is a mission-first entity,
not a standard profit-maximizing company.
