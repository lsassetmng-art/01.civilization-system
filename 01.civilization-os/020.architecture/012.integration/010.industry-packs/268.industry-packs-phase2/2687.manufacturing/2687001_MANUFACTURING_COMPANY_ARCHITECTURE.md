# ============================================================
# 1. PURPOSE
# ============================================================

Defines the official architecture for manufacturing_company.

manufacturing_company is a system-limited company type
that plans, produces, and outputs structured manufactured goods.


# ============================================================
# 2. CORE RULE
# ============================================================

manufacturing_company is system company only.


# ============================================================
# 3. CORE RESPONSIBILITIES
# ============================================================

- production planning
- input/output structure management
- facility-linked manufacturing execution
- batch output handling
- production quality control


# ============================================================
# 4. FINAL RULE
# ============================================================

Manufacturing is a structured production system,
not a retail or design-only system.
