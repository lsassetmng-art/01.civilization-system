# ============================================================
# 1. PURPOSE
# ============================================================

Defines the official architecture for bank.

bank is a system-limited company type
that handles deposit, loan, collateral,
and settlement-linked financial services.


# ============================================================
# 2. CORE RULE
# ============================================================

bank is system company only.
User flow may not establish bank.


# ============================================================
# 3. CORE RESPONSIBILITIES
# ============================================================

- account management
- deposit and withdrawal
- lending and repayment
- collateral linkage
- delinquency handling
- settlement support


# ============================================================
# 4. FINAL RULE
# ============================================================

bank is a core system financial intermediary.
