# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for initial infrastructure placement
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

Initial infrastructure placement must reflect:
- nation identity
- city role
- population scale
- zoning posture
- geography and disaster profile
- industry and logistics needs

Infrastructure is not generic utility only.
It is part of national identity and world function.


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each nation and city infrastructure seed should define:
- transport infrastructure
- utility infrastructure
- logistics and storage infrastructure
- emergency and resilience infrastructure


# ============================================================
# 4. FINAL RULE
# ============================================================

Infrastructure seed is a first-class layer
that binds economy, housing, and governance together.
