# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for road,
rail,
station,
and port placement and governance
inside Civilization.

This architecture defines:

- infrastructure placement on field map
- line and node distinction
- nation-type default governance
- nation-specific override governance
- operator class restrictions
- infrastructure authority and permission
- station and port node control


# ============================================================
# 2. CORE PRINCIPLE
# ============================================================

Infrastructure must be determined in two layers:

- nation type default governance
- nation-specific override governance

Existing nations
must not be reduced
to infrastructure behavior
derived from nation type alone.


# ============================================================
# 3. OPERATIONAL PRINCIPLE
# ============================================================

Infrastructure must separate:

- policy and governance
- placement authority
- line placement
- node placement
- operational control
- public versus delegated operator role


# ============================================================
# 4. FINAL RULE
# ============================================================

Infrastructure placement
must be public, delegated, or licensed
by clear governance rule,
not by open generic building rights.
