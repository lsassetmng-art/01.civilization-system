# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical enum master
for placeable infrastructure inside Civilization.

This architecture fixes the candidate value groups
that shape:

- infrastructure type
- line versus node behavior
- operator class
- authority scope
- placement and approval behavior
- control posture
- strategic and cross-border behavior


# ============================================================
# 2. CORE PRINCIPLE
# ============================================================

Infrastructure must not be treated
like ordinary placeable facilities.

Infrastructure requires explicit candidate values
because it affects:

- public authority
- field map execution
- permission gating
- replacement and removal risk
- strategic control
- nation and city differences


# ============================================================
# 3. FINAL RULE
# ============================================================

Placeable infrastructure
must be governed by explicit enum masters
before implementation behavior is fixed.
