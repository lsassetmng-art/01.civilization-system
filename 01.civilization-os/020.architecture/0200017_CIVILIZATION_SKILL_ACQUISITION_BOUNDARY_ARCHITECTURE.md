# PURPOSE

Define the official boundary between Civilization-side
learning opportunities and PersonaOS-side skill truth.


# RULE

CivilizationOS may define:

- training
- education
- work experience
- practice session
- learning event
- skill-acquisition opportunity

These may trigger Persona skill gain or skill growth.


# TRUTH BOUNDARY

CivilizationOS is the source of:
- learning opportunity
- training event
- education event
- work-experience event

PersonaOS is the source of:
- skill definition master
- Persona-owned skill truth
- skill level state
- skill capability exposure


# CONSTRAINT

CivilizationOS must not become
the source of Persona skill truth.

Civilization-side learning events must resolve
into PersonaOS-defined skills.


# FINAL DEFINITION

CivilizationOS provides world-side opportunities
for skill gain and growth.
PersonaOS remains the sole owner
of skill definition and Persona-owned skill truth.
