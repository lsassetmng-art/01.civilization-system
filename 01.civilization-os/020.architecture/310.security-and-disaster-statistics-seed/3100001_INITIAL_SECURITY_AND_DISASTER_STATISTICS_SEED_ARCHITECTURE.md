# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for nation-level and city-level security statistics,
crime exposure statistics,
accident statistics,
and disaster vulnerability statistics
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

Security and disaster statistics must reflect:
- nation identity
- public order posture
- policing posture
- welfare and poverty structure
- industrial exposure
- transport and port exposure
- sea and climate exposure
- urban density and infrastructure quality


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each nation seed should define:
- crime tendency posture
- fraud and property crime posture
- violent incident posture
- industrial and transport accident posture
- maritime accident posture
- disaster vulnerability posture
- internal city and class gap posture


# ============================================================
# 4. FINAL RULE
# ============================================================

Security and disaster statistics seed must make visible
how danger is distributed,
how order is preserved or fails,
and how geography becomes risk.
