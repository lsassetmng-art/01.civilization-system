# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for initial real-estate facilities,
housing sales facilities,
rental mediation facilities,
property management facilities,
and redevelopment support facilities
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

Real-estate and housing-distribution facilities must reflect:
- nation identity
- housing structure
- class structure
- rental vs ownership posture
- redevelopment posture
- mixed-use posture
- city role and density


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each nation and city seed should define:
- housing sales naming posture
- rental mediation naming posture
- property management naming posture
- redevelopment support naming posture
- city-level assignment


# ============================================================
# 4. FINAL RULE
# ============================================================

Real-estate seed must make visible
how people obtain, rent, manage, and rebuild housing.
