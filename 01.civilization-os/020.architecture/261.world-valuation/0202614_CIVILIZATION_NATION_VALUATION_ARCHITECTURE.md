# ============================================================
# CIVILIZATION NATION VALUATION ARCHITECTURE
# ============================================================

status: canonical
layer: 020.architecture
system: civilization-os
owner: Boss
prepared_by: Zero

purpose:
Define the official nation-level valuation personality
used by CivilizationOS real-estate and geography-aware economy.

This architecture translates Foundation world truth
about the seven major states into valuation behavior.

It does not define legal property regime.
It defines nation-specific valuation personality
within the world-common geography layer.


# ============================================================
# 1. CORE PRINCIPLE
# ============================================================

Nation valuation must be derived from:

- Foundation world truth
- continent valuation personality
- national role in civilization
- national economic emphasis
- national military/education/trade/culture identity

Core principle:

continent gives macro bias  
nation gives civilization-role bias  
district and property give local price shape

Nation valuation must not replace property regime legality.
That remains a separate nation-regime layer.

# ============================================================
# 9. NATION DIFFERENTIATION RULE
# ============================================================

The seven states must not collapse into the same valuation pattern.

Key anchors:

- Helios = industry/resource premium
- Nova = finance/commercial premium
- Seiwa = education/infrastructure premium
- Gladia = strategic/military premium
- Orpheus = maritime/cultural premium
- Aurelia = legacy-distortion/redevelopment premium
- Free Federation = flexible mixed asymmetry premium

This differentiation is mandatory.


# ============================================================
# 10. RELATION TO CONTINENT VALUATION
# ============================================================

Nation valuation modifies continent valuation personality.

Examples:
- Nova intensifies central continent commercial/prestige premium
- Seiwa intensifies central continent education/infrastructure premium
- Gladia intensifies central continent strategic/restricted premium
- Helios intensifies north continent industrial/resource premium
- Orpheus intensifies south continent maritime/cultural premium
- Aurelia intensifies south continent redevelopment distortion
- Free Federation intensifies south continent mixed flexibility

Nation valuation must remain subordinate to world truth.


# ============================================================
# 11. RELATION TO DISTRICT AND PROPERTY VALUE
# ============================================================

District and property valuation should derive from:

- continent valuation bias
- nation valuation bias
- city role
- district role
- property use class
- infrastructure and scarcity

Nation valuation is not the final price by itself,
but a strong multiplier on local valuation personality.


# ============================================================
# 12. REGIME SEPARATION RULE
# ============================================================

Nation valuation is not the same as nation property regime.

Examples of what must remain outside this layer:
- ownership legality
- transfer approval requirement
- foreign holding legality
- expropriation power
- company holding legality

Those belong to real-estate rights regime layers.


# ============================================================
# 13. FINAL ARCHITECTURAL RULE
# ============================================================

CivilizationOS nation valuation must reflect Foundation truth.

Core summary:

- each nation has distinct valuation personality
- nation valuation sits between continent and district layers
- nation valuation affects price tendency, not legal regime
- district and property valuation should consume nation valuation bias
