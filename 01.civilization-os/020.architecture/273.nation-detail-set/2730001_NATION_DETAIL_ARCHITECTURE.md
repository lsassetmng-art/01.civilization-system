# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical detail architecture
for major nations of Civilization.

This layer refines nation seed design beyond
basic profile and placement.


# ============================================================
# 2. DETAIL DIMENSIONS
# ============================================================

Each major nation detail must define:
- nation type
- government model
- power structure
- city structure
- industry structure
- education posture
- military posture
- public service posture
- market posture
- symbolic identity


# ============================================================
# 3. FINAL RULE
# ============================================================

Major nations must be explicit world entities,
not only names plus capitals.
