# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for initial education institution placement
and medical institution placement
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

Education and medical placement must reflect:
- nation identity
- city role
- population structure
- housing structure
- transport access
- ideology and religion posture
- island or frontier constraints

This layer defines structured initial presence,
not parcel-level final implementation.


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each nation and city seed should define:
- child education baseline
- higher education baseline
- vocational and training baseline
- doctrinal or ritual schooling if applicable
- general hospital baseline
- clinic baseline
- emergency medical baseline
- special national naming rules


# ============================================================
# 4. FINAL RULE
# ============================================================

Education and medical seed is a first-class
civilization layer because it drives
population growth, labor quality, welfare,
and life-event realism.
