# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for corporate hierarchy,
meeting culture,
reporting culture,
promotion culture,
turnover culture,
and internal political behavior
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

Corporate organization culture must reflect:
- nation identity
- class structure
- work hierarchy culture
- hiring culture
- communication culture
- competition culture
- military or civic discipline
- family-business and ownership posture


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each nation seed should define:
- reporting posture
- meeting posture
- hierarchy posture
- promotion posture
- turnover posture
- internal politics posture
- family-business influence posture


# ============================================================
# 4. FINAL RULE
# ============================================================

Corporate culture seed must make visible
how power, advancement, obedience, and conflict
operate inside organizations.
