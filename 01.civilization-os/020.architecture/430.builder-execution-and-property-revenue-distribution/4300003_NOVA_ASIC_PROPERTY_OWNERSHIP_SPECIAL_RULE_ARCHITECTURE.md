# PURPOSE

Define the special ownership rule
for Nova and ASIC City property structures.


# SPECIAL OWNERSHIP RULE

In Nova,
the following owners are ASIC:

- owner of multi-tenant corporate buildings
  that contain multiple world companies
- storefront property owner in ASIC City
- employee dormitory owner


# INTERPRETATION RULE

This ownership rule means that
tenant,
operator,
occupant,
leaseholder,
and assigned resident
must be separated from owner.

Using or operating a property
does not imply ownership.


# REVENUE EFFECT RULE

Under this special rule,
property-origin income in the covered categories
flows to ASIC as owner-side income,
unless another explicit contract overrides it.


# FINAL RULE

Nova property categories covered by this special rule
must treat ASIC as owner,
even when operation or occupancy belongs to another actor.
