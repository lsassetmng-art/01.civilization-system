# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for initial public facility placement,
school placement, training placement,
and residential-belt placement
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

Initial public-facility placement must reflect:
- nation identity
- city role
- population size
- zoning posture
- housing structure
- historical and political traces

This layer defines structured initial presence,
not parcel-level exact placement.


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each city seed should define:
- government facility baseline
- school baseline
- medical baseline
- public square / park baseline
- religious / ritual baseline
- training facility baseline
- transport hub baseline
- residential belt posture


# ============================================================
# 4. FINAL RULE
# ============================================================

Public facilities and residential belts
must be city-aware and nation-aware from the start.
