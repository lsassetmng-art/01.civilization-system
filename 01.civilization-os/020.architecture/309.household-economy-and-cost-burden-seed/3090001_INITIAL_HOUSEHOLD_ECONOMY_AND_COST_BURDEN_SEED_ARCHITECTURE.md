# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical architecture
for household taxes,
living costs,
housing costs,
education costs,
medical costs,
and recurring household burden
in Civilization seed-world planning.


# ============================================================
# 2. CORE RULE
# ============================================================

Household economy and cost burden must reflect:
- nation identity
- welfare posture
- labor-market posture
- education posture
- housing posture
- medical support posture
- family formation timing
- urban and class inequality

These are not abstract prices only.
They must explain how ordinary households live,
struggle, save, and fail.


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each nation seed should define:
- tax burden posture
- household fixed-cost posture
- housing burden posture
- education burden posture
- medical burden posture
- family-raising burden posture
- class and city gap posture


# ============================================================
# 4. FINAL RULE
# ============================================================

Household economy seed must make visible
how income becomes survival,
mobility, savings, and family sustainability.
