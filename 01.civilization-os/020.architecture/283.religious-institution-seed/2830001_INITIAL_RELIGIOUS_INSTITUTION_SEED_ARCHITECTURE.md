# 2. CORE RULE
# ============================================================

Religious institution placement must reflect:
- Foundation-derived theology
- nation identity
- city role
- ritual density
- political relationship with religion
- education and welfare linkage

Religious facilities are not only symbolic.
They may affect:
- life events
- education
- welfare
- memorial practice
- political legitimacy
- inter-nation tension


# ============================================================
# 3. REQUIRED DIMENSIONS
# ============================================================

Each religious seed should define:
- major religious corporations
- headquarters placement
- cathedral / temple / shrine hierarchy
- doctrinal school linkage
- welfare and charity linkage
- city-level placement baseline
- faith population scale


# ============================================================
# 4. FINAL RULE
# ============================================================

Religious institution seed is a first-class civilization layer
because it shapes ceremony, ideology, welfare, and conflict.
