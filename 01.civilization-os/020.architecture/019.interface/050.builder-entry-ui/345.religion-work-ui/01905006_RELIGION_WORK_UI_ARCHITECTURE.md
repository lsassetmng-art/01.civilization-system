# ============================================================
# 1. PURPOSE
# ============================================================

Defines the canonical UI architecture
for religion work,
including ritual operation,
festival preparation,
donation handling,
believer care,
pilgrimage operation,
doctrinal outreach,
and religious institution continuity.


# ============================================================
# 2. CORE PRINCIPLE
# ============================================================

Religion Work UI must separate:

- doctrine and institution context
- ritual scheduling
- clergy and staff assignment
- donation and resource management
- believer support and participation
- pilgrimage and sacred-site handling
- outreach and influence handling
- memorial and history persistence

No religion UI should reduce religion
to a simple blessing action
without resource, community,
or institutional continuity.


# ============================================================
# 3. SUPPORTED RELIGION OPERATION FAMILIES
# ============================================================

supported_religion_operation_families:
- local_worship_operation
- major_temple_or_cathedral_operation
- pilgrimage_site_operation
- festival_religion_operation
- donation_and_charity_operation
- doctrine_outreach_operation
- memorial_and_funeral_operation
- state-linked_religious_operation


# ============================================================
# 4. REQUIRED UI LAYERS
# ============================================================

required_ui_layers:
- institution_overview_layer
- ritual_schedule_layer
- clergy_and_staff_layer
- donation_and_resource_layer
- believer_and_community_layer
- pilgrimage_and_site_layer
- outreach_and_doctrine_layer
- crisis_and_scandal_layer
- ai_assist_layer
- result_summary_layer
- religion_history_layer


# ============================================================
# 5. FINAL RULE
# ============================================================

Religion UI must make religious work feel
communal, symbolic,
resource-bound, and politically meaningful.
