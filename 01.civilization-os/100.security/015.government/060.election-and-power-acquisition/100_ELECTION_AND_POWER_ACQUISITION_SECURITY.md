# 1. PURPOSE

Defines security controls for elections and power acquisition.

# 2. SECURITY SCOPE

- election/candidate integrity
- vote-result integrity
- transfer authorization
- election audit protection

# 3. SECURITY RULES

- candidate registration and vote result must be attributable
- power transfer requires explicit authorization and record
- hidden power transfer must be detectable
- audit output must preserve election identity
