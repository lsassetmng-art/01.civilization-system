# 1. PURPOSE

Defines security controls for reward grants.

# 2. SECURITY SCOPE

- grant authorization
- source/target provenance
- grant-state integrity
- grant audit protection

# 3. SECURITY RULES

- grant requires explicit authorization
- source and target scope must remain attributable
- applied state must be tamper-evident
- unauthorized grant misuse must be detectable
