## 3. Required Decision Update Questions

Every decision update must answer:

1. Which current field changes?
2. Why does it change?
3. Which prior decision is superseded?
4. Does it affect:
   - phase1 close posture
   - execution resume posture
   - transition posture
   - phase2 entry posture

## 4. Hard Rule

A future decision change should start from this template
before it becomes a canonical update document.
