## 1. Purpose

Defines the minimum document set required before any future posture change.

## 2. Minimum Change Control Set

Before any future change, the operator must consult at minimum:

- final decision pack
- decision revision rule
- the correct reconsideration or update template
- decision update ledger schema
- canonical baseline snapshot

## 3. Minimum Change Rule

No future posture change should be attempted
without this minimum set being consulted together.

## 4. Hard Rule

A single memo or single template alone is not enough
to justify or record a valid posture change.

## 5. Acceptance Checklist

- minimum set fixed
- minimum change rule fixed
- no-single-doc-change rule fixed
