## 1. Purpose

Defines the minimum portable reference bundle for future transport across sessions.

## 2. Minimum Portable Bundle

The minimum portable bundle is:

- current posture one-page
- canonical posture handoff
- canonical baseline snapshot
- final decision pack integrated status
- operator entrypoint master
- no-change default posture rule

## 3. Portable Bundle Rule

If future work must continue in another session or by another operator,
start with this minimum portable bundle first.

## 4. Extended Bundle Rule

If governed change is intended, extend the bundle with:

- decision revision rule
- correct request template
- minimum change control set
- decision update ledger schema

## 5. Hard Rule

Do not export only one document and assume continuity is preserved.
