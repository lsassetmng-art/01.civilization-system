## 1. Purpose

Provides a compact operator checklist for future handling.

## 2. Quick Checklist

Before doing anything, check:

1. read canonical posture handoff
2. confirm frozen baseline snapshot
3. decide whether the intent is:
   - no change
   - reopen
   - resume reconsideration
   - phase2 entry reconsideration
   - decision update
4. if no change, stop
5. if change is intended, use governance and template path
6. re-run audit only when a valid trigger exists

## 3. Compact Rule

When uncertain, prefer:
- read-only posture
over
- improvised change

## 4. Hard Rule

Do not start from old detailed packs or partial execution records first.
