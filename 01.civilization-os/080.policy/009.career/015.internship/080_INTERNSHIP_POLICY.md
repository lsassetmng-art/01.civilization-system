# 1. PURPOSE

Defines policy rules for internship structures.

# 2. POLICY RULES

- host scope, participant scope, and slot identity must be explicit
- one explicit outcome state must exist per finalized participation
- hidden internship-outcome mutation is prohibited
- implicit participation or outcome publication without resolved program context is prohibited

# 3. FAILURE RULE

- missing host or participant scope must reject
- invalid slot or participation binding must reject
- hidden internship-outcome mutation must reject
