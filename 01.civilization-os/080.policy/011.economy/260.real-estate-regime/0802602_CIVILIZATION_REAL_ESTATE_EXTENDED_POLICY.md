Define policy for extended real-estate domains inside CivilizationOS.

This includes:
- tax
- inheritance
- transfer
- auction
- finance
- persona housing screening

All must be nation-governed and DB-controlled.


# ============================================================
# 1. CORE POLICY
# ============================================================

The system must provide common structures
while leaving operational legality and thresholds
to nation DB rules.


# ============================================================
# 2. FAIL-CLOSED POLICY
# ============================================================

The system must fail closed if:
- a mandatory nation rule is missing
- rule resolution is contradictory
- registry finality cannot be reached
- tax, transfer, auction, finance, or screening rule chain is incomplete


# ============================================================
# 3. SCREENING POLICY
# ============================================================

Housing screening decisions must preserve:
- source facts
- calculated screening state
- screening snapshot
- explainable blocking reason


# ============================================================
# 4. FINAL POLICY RULE
# ============================================================

Extended real-estate policy in CivilizationOS must be:
- nation-governed
- DB-controlled
- snapshot-preserving
- fail-closed
