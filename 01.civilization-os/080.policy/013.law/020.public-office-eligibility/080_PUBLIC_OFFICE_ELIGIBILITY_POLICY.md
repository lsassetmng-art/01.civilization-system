# 1. PURPOSE

Defines policy rules for public office eligibility.

# 2. POLICY RULES

- office scope, candidate scope, and requirement set must be explicit
- one explicit result must exist per evaluation
- hidden eligibility override is prohibited
- implicit approval without evaluation is prohibited

# 3. FAILURE RULE

- missing office scope must reject
- invalid requirement set must reject
- hidden eligibility override must reject
