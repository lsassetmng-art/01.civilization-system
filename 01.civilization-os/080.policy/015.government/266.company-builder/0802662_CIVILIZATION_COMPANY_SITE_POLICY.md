purpose:
Define policy for company core/site/opening handling
inside CivilizationOS.


# ============================================================
# 1. CORE POLICY
# ============================================================

A company must be able to exist with real operational sites
inside a nation.

Company core and company site must remain distinct.


# ============================================================
# 2. OPENING POLICY
# ============================================================

Store or site opening:
- is allowed only in nation-designated allowed zones
- does not require prior nation application
- must generate nation notification after becoming effective

This is mandatory.


# ============================================================
# 3. PROPERTY TRANSACTION POLICY
# ============================================================

Any company type may:
- buy property
- lease property
- resell property
- lease tenant units

However:
transaction execution must be mediated
through the system real-estate company.

This is mandatory.


# ============================================================
# 4. FAIL-CLOSED POLICY
# ============================================================

The system must fail closed if:
- site compatibility is unresolved
- target zone compatibility is unresolved
- occupancy basis is unresolved
- nation notification chain is unresolved where required
- property transaction bypasses system real-estate mediation


# ============================================================
# 5. FINAL POLICY RULE
# ============================================================

Company site policy in CivilizationOS must be:
- site-aware
- zone-aware
- notification-aware
- mediation-aware
- fail-closed
