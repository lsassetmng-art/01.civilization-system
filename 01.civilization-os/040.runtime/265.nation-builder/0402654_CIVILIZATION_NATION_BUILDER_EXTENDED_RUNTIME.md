# ============================================================
# 1. RUNTIME ROLE
# ============================================================

The runtime must evaluate:
- template applicability
- symbolic identity completeness
- governance structure completeness
- major city role consistency
- population composition consistency
- priority industry validity
- value/culture/religion profile completeness
- public service baseline completeness
- holiday/event profile validity
- operating profile validity
- initial frame completeness
- notification profile completeness


# ============================================================
# 2. FAIL-CLOSED RULE
# ============================================================

The runtime must fail closed if:
- required template relation is unresolved where used
- governance structure is unresolved
- major city role assignment is unresolved where required
- population composition is contradictory
- public service baseline is unresolved
- initial frame is unresolved where required for readiness
- notification profile is unresolved where required


# ============================================================
# 3. FINAL RUNTIME RULE
# ============================================================

Extended nation builder runtime must be:
- template-aware
- identity-aware
- governance-aware
- city-role-aware
- social-profile-aware
- service-aware
- event-aware
- notification-aware
- fail-closed
