# ============================================================
# PERSONA GROWTH ARCHITECTURE
# ============================================================

status: canonical
layer: architecture

definition:
Persona growth models long-term personal development.

growth_sources:
- education
- work
- relationships
- adversity
- achievements
- repeated practice

rules:
- growth must identify explicit source events
- growth is gradual, not arbitrary
- growth does not bypass skill, emotion, or memory systems

final_rule:
Persona growth must be explainable from accumulated event history.
