# ============================================================
# IDEOLOGY AND CONTROL ARCHITECTURE
# ============================================================

status: canonical
layer: architecture
domain: civilization

# ============================================================
# DEFINITION
# ============================================================

Nation ideology defines how strongly a nation controls
economy, information, and market behavior.

# ============================================================
# COMPONENTS
# ============================================================

- ideology_type
- market_control_level
- censorship_level

# ============================================================
# CENSORSHIP
# ============================================================

- content must be evaluated before distribution
- restricted content must be blocked or filtered

# ============================================================
# MARKET CONTROL
# ============================================================

- nation may enforce minimum or maximum prices
- nation may fix prices
- nation may subsidize goods

# ============================================================
# RULE
# ============================================================

- control level must be configurable
- censorship must be content-type aware
- market intervention must affect all companies

# ============================================================
# PURPOSE
# ============================================================

- simulate political systems
- create diverse economic environments
- influence player strategy
