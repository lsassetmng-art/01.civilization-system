# ============================================================
# LAW ARCHITECTURE OVERVIEW
# ============================================================

status: canonical
layer: architecture
domain: law

purpose:

Law defines constraints of behavior.

It does NOT execute actions.
It defines what is allowed / forbidden / required.

scope:

- legal entities
- contracts
- liability
- disputes
- penalties
- compliance

principles:

Law is declarative  
Law is enforceable via runtime  
Law is auditable  
Law is immutable after execution
