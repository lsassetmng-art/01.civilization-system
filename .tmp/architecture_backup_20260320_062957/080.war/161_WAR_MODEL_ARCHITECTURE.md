# ============================================================
# WAR MODEL ARCHITECTURE
# ============================================================

status: canonical
layer: architecture

definition:

War is executed as strategic resolution, not real-time combat.

reason:
- prevents micro-control complexity
- keeps focus on high-level decisions
- aligns with role-based gameplay

inputs:
- military_strength
- economy
- logistics
- alliance_support
- intelligence
- morale / loyalty

resolution:
- turn-based or event-based calculation
- outcome determined by weighted factors

visibility:
- limited information to players
- full information only to top command (or system abstraction)

principle:
- war is strategic layer, not action layer
