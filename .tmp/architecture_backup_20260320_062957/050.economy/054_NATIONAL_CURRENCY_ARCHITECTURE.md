# ============================================================
# NATIONAL CURRENCY ARCHITECTURE
# ============================================================

status: canonical
layer: architecture
domain: economy

# ============================================================
# DEFINITION
# ============================================================

National currency is a non-cashable currency
representing participation, contribution, and activity.

# ============================================================
# SOURCES
# ============================================================

National currency can only be generated from:

- activity_reward
- login_bonus
- system_grant
- government_grant

# ============================================================
# PROHIBITIONS
# ============================================================

- conversion from real_money is forbidden
- direct purchase using real_money is forbidden

# ============================================================
# LEDGER
# ============================================================

All currency generation must be recorded:

- amount
- source_type
- reference_id
- timestamp

# ============================================================
# ECONOMIC PRINCIPLE
# ============================================================

Supply must be controlled and traceable.

# ============================================================
# ROLE IN SYSTEM
# ============================================================

- used for internal economy
- not exchangeable to real money
- used for progression, access, and influence
