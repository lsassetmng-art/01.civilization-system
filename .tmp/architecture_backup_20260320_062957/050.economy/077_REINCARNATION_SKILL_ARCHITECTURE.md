# ============================================================
# REINCARNATION SKILL ARCHITECTURE
# ============================================================

status: canonical
layer: architecture
domain: persona

# ============================================================
# DEFINITION
# ============================================================

Skills are preserved across reincarnation
but gradually restored during early age.

# ============================================================
# STRUCTURE
# ============================================================

stored_skill
active_skill

# ============================================================
# RULE
# ============================================================

- stored skills are inherited
- active skills start low
- restoration occurs from age 6 to 10

# ============================================================
# RESTORATION
# ============================================================

- age 6 → partial restore
- age 10 → full restore

# ============================================================
# CONSTRAINT
# ============================================================

- stats are NOT inherited
- only skills are carried over

# ============================================================
# PURPOSE
# ============================================================

- preserve effort
- enable replay value
- maintain realism
