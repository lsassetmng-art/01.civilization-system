# ============================================================
# MARKET SYSTEM ARCHITECTURE
# ============================================================

status: canonical
layer: architecture
domain: civilization

# ============================================================
# DEFINITION
# ============================================================

Market is formed by companies setting prices and personas choosing purchases.

# ============================================================
# STRUCTURE
# ============================================================

- companies create products
- companies set prices
- personas choose purchases

# ============================================================
# RULE
# ============================================================

- same product can have different prices per company
- currency must be defined per product
- market must not enforce fixed pricing

# ============================================================
# EFFECT
# ============================================================

- price competition emerges
- economic diversity increases
- player-driven economy

# ============================================================
# PURPOSE
# ============================================================

- simulate real-world market
- enable player business
- create dynamic economy
