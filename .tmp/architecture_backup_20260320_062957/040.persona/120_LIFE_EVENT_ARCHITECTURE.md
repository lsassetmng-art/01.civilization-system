# ============================================================
# LIFE EVENT ARCHITECTURE
# ============================================================

status: canonical
layer: architecture
domain: civilization

# ============================================================
# DEFINITION
# ============================================================

Life events represent major milestones in a persona's life.

# ============================================================
# TYPES
# ============================================================

- birth
- education
- job
- promotion
- marriage
- child
- illness
- death

# ============================================================
# RULE
# ============================================================

- events must be condition-driven
- events must reflect economy and environment
- events must not be fully controlled by player

# ============================================================
# PURPOSE
# ============================================================

- simulate realistic life progression
- connect economy and personal life
