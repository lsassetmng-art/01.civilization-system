# ============================================================
# OVERVIEW
# ============================================================

The world consists of:

- AI Nations (base nations)
- User Nations (player-created)

# ============================================================
# AI NATIONS
# ============================================================

- system initializes 7 AI nations
- these nations are permanent
- cannot be deleted or collapsed
- act as foundational world entities

# ============================================================
# USER NATIONS
# ============================================================

- users can create new nations
- AI cannot create nations

# ============================================================
# COLLAPSE RULE
# ============================================================

User nations may collapse.

Upon collapse, user must choose:

A. LIQUIDATION
- nation becomes inactive

B. CONTINUATION
- nation remains in degraded state

# ============================================================
# STABILITY RULE
# ============================================================

- AI nations ensure world stability
- no destructive world collapse exists

# ============================================================
# LIMIT
# ============================================================

- total nation count controlled via DB

# ============================================================
# PURPOSE
# ============================================================

- maintain stable world base
- enable dynamic player-driven expansion
