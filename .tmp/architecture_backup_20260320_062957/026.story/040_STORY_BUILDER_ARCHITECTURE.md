# ============================================================
# STORY BUILDER ARCHITECTURE
# ============================================================

status: canonical
layer: architecture
domain: story

# ============================================================
# DEFINITION
# ============================================================

Story Builder converts a snapshot into
a sequence of playable scenes.

# ============================================================
# INPUT
# ============================================================

- snapshot

# ============================================================
# OUTPUT
# ============================================================

- ordered scene list

# ============================================================
# SCENE MODEL
# ============================================================

Scene includes:

- type
- background
- characters
- content
- choices
- next links

# ============================================================
# SCENE TYPES
# ============================================================

- dialogue
- choice
- explore
- puzzle

# ============================================================
# TRANSFORMATION RULES
# ============================================================

event → dialogue scene  
entity → character  
location → background  

# ============================================================
# FLOW
# ============================================================

snapshot
↓
event extraction
↓
scene generation
↓
scene linking

# ============================================================
# BRANCHING
# ============================================================

- branching allowed in story
- branching must not affect world
- final convergence required

# ============================================================
# DETERMINISM
# ============================================================

same snapshot → same story

# ============================================================
# FINAL RULE
# ============================================================

Story must be a deterministic transformation
of snapshot data.
