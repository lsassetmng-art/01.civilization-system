# PRINCIPLE

Civilization systems must remain stoppable.


# IMPLEMENTATION PRINCIPLE

Systems must support runtime control flags:

os_enabled
dispatch_enabled
ingest_enabled
apply_enabled


# CONSTRAINTS

Unstoppable systems are prohibited.
