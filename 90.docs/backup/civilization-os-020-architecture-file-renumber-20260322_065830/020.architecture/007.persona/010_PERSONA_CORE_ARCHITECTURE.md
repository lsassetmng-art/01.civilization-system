# ============================================================
# DEFINITION
# ============================================================

Persona is the core life entity used across Civilization.

A persona consists of:

- identity
- age
- traits
- skills
- stats
- memory
- relationships
- owned assets
- life state

# ============================================================
# RULE
# ============================================================

- persona is persistent
- persona state changes only through defined systems
- persona is usable across compatible domains
