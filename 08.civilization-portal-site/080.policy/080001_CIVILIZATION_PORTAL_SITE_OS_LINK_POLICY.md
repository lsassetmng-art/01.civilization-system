# 1. 文書目的
# ============================================================

本書は、OS間リンク原則を定義する。

# ============================================================
# 2. 原則
# ============================================================

- OS間の直接リンクは禁止する。
- 各OSから Civilization Portal Site への戻りは許可する。
- 認証切れ時の CivilizationOS ログインページへの遷移は条件付きで許可する。
