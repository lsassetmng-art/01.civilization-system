# ============================================================
# 2. OSカード定義
# ============================================================

OSカードは、
Civilization Portal Site 上で各OSを紹介し、
状態を示し、
詳細確認またはWeb入口への遷移へ導く標準表示単位とする。

# ============================================================
# 3. 共通要素
# ============================================================

- OS名
- 見出し
- 一言説明
- 主用途要約
- 利用条件要約
- 状態バッジ
- 詳細導線
- 遷移導線

# ============================================================
# 4. 利用箇所
# ============================================================

- OS一覧
- OS詳細要約部
- マイランチャー
- 特集枠
- 管理画面プレビュー
