# 1. 文書目的
# ============================================================

本書は、Portal Site におけるログイン要求フローを定義する。

# ============================================================
# 2. フロー
# ============================================================

1. 利用者が対象OSへの遷移を要求する。
2. Portal Site はログイン要否を確認する。
3. 未ログインであれば CivilizationOS のWebログインページへ誘導する。
4. 認証成功後、復帰文脈に基づき対象OS Web入口へ戻す。
