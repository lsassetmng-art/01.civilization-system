# ============================================================
# 1. FINAL DECISION
# ============================================================

PersonaOS will align with the same structural discipline
used for other OS domains as much as possible.

Therefore:

900.society is not used as a canonical layer.
910.civilization is not used as a canonical layer.

Meaning-oriented explanations are placed in 920.meta.
Cross-system connection meaning is placed in 140.integration.
Non-negotiable existence principles are placed in 010.constitution.

# ============================================================
# 2. REASON
# ============================================================

The goal is to reduce unnecessary structural difference
between PersonaOS and other OS domains.

A separate society-facing or civilization-facing layer
would make PersonaOS structurally more special
than necessary.

PersonaOS should remain structurally disciplined.

# ============================================================
# 3. CANONICAL PLACEMENT RULE
# ============================================================

Use 010.constitution for:
- top-level existence principles
- truth authority principles
- non-negotiable role principles

Use 140.integration for:
- BusinessOS relation
- StreamingOS relation
- PocketSecretary relation
- Civilization-linked connection meaning
- external system connection contracts

Use 920.meta for:
- design intent explanation
- reading guidance
- structural interpretation
- world/meaning explanation
- migration and review records

# ============================================================
# 4. PROHIBITED INTERPRETATION
# ============================================================

It is prohibited to treat:
- 920.meta as a runtime layer
- 140.integration as authority origin
- 010.constitution as a place for excessive implementation detail

# ============================================================
# 5. HANDLING OF OLD STRUCTURE
# ============================================================

If 900.society or 910.civilization exists,
they should be treated as legacy or transitional structure.

Their meaningful contents should be reassigned as follows:

social/explanatory meaning
-> 920.meta

cross-system/civilization connection meaning
-> 140.integration

non-negotiable existence principle
-> 010.constitution

# ============================================================
# 6. FINAL POSITION
# ============================================================

PersonaOS keeps structural consistency
with other OS domains by avoiding
special top-level meaning layers.

Meaning is preserved,
but placement is normalized.
