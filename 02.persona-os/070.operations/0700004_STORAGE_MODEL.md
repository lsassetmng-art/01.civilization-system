# PURPOSE

Define how PersonaOS stores data and assets.

# STORAGE TYPES

Database storage

Used for:

persona state
events
growth logs
snapshots

Object storage

Used for:

visual assets
generated images
imported resources

# STORAGE PRINCIPLE

Structured data must remain in database.
Binary assets must reside in object storage.
