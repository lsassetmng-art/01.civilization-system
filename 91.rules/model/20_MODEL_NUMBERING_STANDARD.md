# ============================================================
# PURPOSE
# ============================================================

Define the canonical numbering system
for all Civilization system models.


# ============================================================
# NUMBER RANGE
# ============================================================

0000–0999   rules

1000–1999   foundation models

2000–3999   civilization models

4000–4999   persona models

5000–5999   business models

6000–6999   life models

7000–7999   game models

8000–8999   streaming models


# ============================================================
# PRINCIPLES
# ============================================================

Model numbers must be unique.

Models must remain within
their designated OS domain range.

Cross-domain reuse is prohibited.


# ============================================================
# EXTENSION RULE
# ============================================================

Each OS is allocated
1000 model slots minimum.

Additional expansion
must extend the upper bound
without breaking existing numbers.


# ============================================================
# FINAL RULE
# ============================================================

Model numbers represent
the structural namespace
of Civilization architecture.

They must remain stable
once published.
