# AIWorkerOS Policy Append: Series Public Directory

status: active
phase: series public directory
scope: AIWorkerOS only

## Policy

The series public directory may expose:

- manufacturer name
- series name
- series summary
- series tendency on three canonical axes
- public app reference metadata

The directory must not expose:

- private cross-company comparison
- unpublished performance equivalence
- hidden ranking
- safety-boundary weakening claims
- action restriction bypass

## Minimum policy note

For series with action_restriction = minimum_policy, minimum does not mean unsafe.

It still blocks illegal, destructive, harmful, contract-outside, or unauthorized actions.
