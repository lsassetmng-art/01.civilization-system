# AIWorkerOS Model Append: Series Public Directory

status: active
phase: series public directory
scope: AIWorkerOS only

## Purpose

Define a common public directory for AI worker series.

This directory lets applications read HD, LoVerS, Beyond, and MEGAMI series in one format.

## Scope

Included series:

- HD Series / HDシリーズ
- LoVerS Series / LoVerSシリーズ
- Beyond Series / Beyondシリーズ
- MEGAMI Series / MEGAMIシリーズ

## Three-axis tendency

The directory exposes the canonical three series tendency axes:

- initiative / 積極性
- user_influence / ユーザー影響度
- action_restriction / 行動制限

## Series tendency summary

HD Series:

- 積極性: 中
- ユーザー影響度: 変化なし
- 行動制限: 指定規約厳守

LoVerS Series:

- 積極性: 各個体による
- ユーザー影響度: やや変化
- 行動制限: 指定規約厳守

Beyond Series:

- 積極性: 高
- ユーザー影響度: ユーザーの悪特性を補完する
- 行動制限: 指定規約厳守

MEGAMI Series:

- 積極性: 低
- ユーザー影響度: 変化なし
- 行動制限: 最低限

## No comparison rule

The public directory does not publish cross-company performance comparisons.

Do not expose private comparative claims such as "series A is equivalent to series B".
