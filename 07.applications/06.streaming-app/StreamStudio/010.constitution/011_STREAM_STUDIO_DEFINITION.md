# 1. DEFINITION

StreamStudio is a StreamingOS creator production,
publishing, commerce, and membership management application
operating under 07.applications/06.streaming-app.

It is not only an upload screen.
It is a creator operations cockpit that unifies:
- upload
- edit preparation
- review
- approval
- collaboration
- publication
- external push extension
- revenue split preparation
- Civilization marketplace listing
- paid video access setting
- membership program management

# 2. POSITIONING

- StreamingOS is the platform truth holder for creator asset and publication.
- StreamStudio is the creator-side production and commerce app.
- StreamWatch is the viewer-side watch app.
- Civilization Marketplace is the commerce distribution surface.
- External platforms are downstream publication targets.

# 3. MAIN USERS

- creator
- collaborator
- producer
- administrator
- membership_operator
