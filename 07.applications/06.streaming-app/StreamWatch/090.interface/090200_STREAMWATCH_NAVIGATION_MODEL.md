## 1. Purpose

This document fixes navigation structure across device classes.

## 2. Mobile Primary Navigation

The mobile primary navigation is:

- Home
- Category
- Search
- Library
- Following

## 3. Library Sections

Library contains:

- Favorites
- Watch Later
- History
- Playlists
- Purchased / Entitled section

## 4. Tablet and Desktop

Tablet and desktop may expose broader visible navigation and side-entry surfaces because more space is available.

## 5. TV Navigation

TV-oriented navigation should favor:

- Home
- Live
- Category
- Library
- Search
- Settings

## 6. Important Rule

Mobile must not explode into too many bottom-navigation entries.
