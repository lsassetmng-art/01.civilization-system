# PURPOSE

Define the role of the application layer.


# DEFINITION

Applications are user-facing products
built on top of one or more OS layers.

Examples:

PocketSecretary
TaskManager
MiniCRM
Journal
FinanceLite


# RELATION TO OS

Applications may use:

civilization-os
persona-os
business-os
life-os
game-os
streaming-os

Applications must not redefine OS responsibilities.
