# CareerLaunch VALUE PROPOSITION
## User value
- 応募単位で書類・面接準備・お礼を束ねられる。
- 日本式履歴書と職務経歴書に強い。
- マルチデバイスで同じ応募状態を追える。
- 面接直前モードで短時間確認ができる。

## Differentiation
- resume builder 単体ではなく execution assistant である。
- job tracker 単体ではなく documents and interview prep を内包する。
- チェックリスト実行を前面に出す。
- ペルソナ / 背景により継続利用の心理ハードルを下げる。
