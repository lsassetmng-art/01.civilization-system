# CareerLaunch CORE PRINCIPLES

app: CareerLaunch
## Principles
- 転職応募実行を最短距離で前に進める。
- 抜けもれ防止を文書生成より優先する。
- 応募ごとに文書と面接準備を分岐できる。
- 求人票貼り付けを正規入力とし、API依存を持ち込まない。
- モバイルではオフライン下書きを最優先する。
- ペルソナ / 背景表示は全プラットフォームで維持する。
- ユーザーの実データは owner 正本で扱い、life スキーマに閉じる。

## Owner promise
このアプリは転職の準備漏れを減らし、応募前・面接前・面接後の行動を checklist と revision で支える。
