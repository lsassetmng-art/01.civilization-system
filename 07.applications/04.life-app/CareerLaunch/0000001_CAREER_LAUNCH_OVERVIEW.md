## One line
CareerLaunch は、履歴書作成から面接後のお礼まで、転職活動を抜けもれなく進めるための独立支援アプリである。

## Primary goal
履歴書から面接後のお礼までの準備を、素早く、抜けもれなく終わらせること。

## Fixed scope
- 求人票貼り付け方式
- 基本プロフィール整理と職務経歴整理
- 履歴書 / 職務経歴書 / 志望動機 / 自己PR / 転職理由
- 応募先管理と書類バージョン管理
- 面接準備チェックリスト、想定質問、回答メモ、逆質問
- 面接後メモ、お礼文、送付確認
- PDF / DOCX 出力
- ペルソナ表示 / 背景表示 / 変更

## Platform
iPhone / Android / PC / Tablet を正式対象とし、PC・タブレットは文書作成、スマホは面接直前確認と即時メモを重視する。

## Schema
本アプリは life スキーマを正本とする。ERP 連携は必須ではなく、求人サイト API にも依存しない。
