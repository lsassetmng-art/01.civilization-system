# CareerLaunch COMMON COMPONENT CANDIDATES
## Strong candidates
- Document Template Common: 履歴書 / 職務経歴書 / 志望動機 / 自己PR / お礼文のテンプレート骨格共通
- Revision History Common: 文書比較、差分要約、提出版固定、復元の共通
- Checklist Execution Common: pre_application / pre_interview / post_interview の共通実行器
- Status Pipeline Common: 興味あり〜内定 / 見送り / 辞退のステータス管理共通
- Offline Draft Queue Common: local save / pending sync / conflict visible の共通
- Export Common: pdf/docx 出力ジョブと frozen snapshot 共通
- Multilingual Label Common: UIラベルと文書テンプレートの言語切替共通
- Multicurrency Display Common: amount + currency の表示・比較共通
- Persona and Background Display Common: ペルソナ表示、背景表示、変更、軽量化の共通
- Advisor Share Common: viewer / editor 権限共有、コメント導線、revocation 共通

## Why now
CareerLaunch は文書・応募・面接準備が混ざるため、上記共通部品を先に切り出す価値が高い。
