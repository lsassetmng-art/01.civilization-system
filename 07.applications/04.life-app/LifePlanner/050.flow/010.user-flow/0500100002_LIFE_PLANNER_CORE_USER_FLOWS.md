# ============================================================
# LIFE PLANNER CORE USER FLOWS
# ============================================================

create_plan_flow:
  - 人生計画表を作成する
  - 対象期間を設定する
  - 目標を登録する
  - 年表にイベントを配置する
  - 費用概算を入力する
  - 保存する

review_flow:
  - 計画表を開く
  - 要見直し項目を確認する
  - 必要な変更を実施する
  - review_logを残す

family_share_flow:
  - 共有相手を選ぶ
  - viewer/editor権限を設定する
  - 共有カテゴリを選ぶ
  - 招待する

scenario_compare_flow:
  - シナリオを複数作成する
  - 費用/イベント/リスクを比較する
  - 採用方針を決める
