- AppDevelopmentStudio is not only a file editor or app builder.
- Its durable value is development-control across design readiness,
  implementation blocker visibility, execution review, and build-progress control.
- The differentiator is not simple authoring, but studio-level control
  that keeps implementation readiness and execution risk visible.

- development and build support identity
- existing business operation role
- existing implementation support direction
- existing platform direction
- existing separation of artifact detail and management summary

differentiator_to_make_explicit:
- development lifecycle control
- implementation readiness visibility
- blocker and exception handling
- studio review and decision-support surfacing

- not merely a code editor
- not merely a no-code screen
- not merely a file manager
- positioned as a development control layer that connects
  readiness review, blocker handling, execution routing,
  and studio attention control
