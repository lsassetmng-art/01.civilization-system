summary:
App Development Studio is an independent business-oriented application
for creating, controlling, and delivering another application.

It supports:
- multiple design input modes
- imported design readiness assessment
- full generation and partial generation
- Java / SQL / test generation
- reusable component reuse
- DB connection profile handling
- configurable review / approval / apply
- Git integration
- bug / version / progress management
- multilingual operation
- optional build execution

main_value:
It allows one application’s full design-to-delivery workflow
to be managed inside one product.
