purpose:
Defines role split between smartphone usage and PC usage.

smartphone_primary_scope:
- requirement input
- quick draft request
- light design edit
- proposal confirmation
- approval / rejection
- notification review
- progress review
- light diff review
- billing/subscription review

pc_primary_scope:
- full generation execution
- detailed design review
- large diff review
- Git handling
- SQL confirmation
- artifact comparison
- starter pack confirmation
- build-related work
- final execution review

model_principle:
Device role split is explicit.
Smartphone and PC are complementary, not identical.
