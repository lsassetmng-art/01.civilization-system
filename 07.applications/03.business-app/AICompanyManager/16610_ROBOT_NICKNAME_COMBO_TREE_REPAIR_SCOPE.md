# AICompanyManager robot nickname / combo add / organization tree repair scope

## President
- Configured on AI企業設定.
- Selected from Business-side robot pool/catalog.
- Internal nickname is required or defaulted.
- Display format: 社内通称@President.

## Organization robot assignment
- Done on 組織詳細 / 組織変更.
- Uses combo-box selection from Business-side robot pool/catalog.
- User selects role and robot, enters internal nickname, then presses 追加.
- Multiple assignments are allowed per organization.
- Display format: 社内通称@役割.

## Organization tree
- Display by department.
- Under each department, display organizations.
- Under each organization, display assigned robots.

## Allocation policy
- AICompanyManager is system-use.
- Unlimited allocation.
- No quantity consumption.

## Boundary
Persistent DB integration is not executed.
