# AICompanyManager system-use unlimited robot allocation canon

## Canon
AICompanyManager is a system-use app.

## Robot selection
Users still select which robot/model/candidate to assign from the Business-side robot pool/catalog.

## Allocation
Robot assignment in AICompanyManager is unlimited allocation.

## No quantity consumption
AICompanyManager must not consume:
- contracted_quantity
- available_quantity
- rental quantity
- inventory count

## Placement records
Actual placement is represented as:
- company-level President assignment
- organization-level Manager / Leader / Worker / Reviewer assignment

## Future reuse
This structure may be reusable for Civilization's company builder, but current implementation remains BusinessOS/system-use AICompanyManager.
