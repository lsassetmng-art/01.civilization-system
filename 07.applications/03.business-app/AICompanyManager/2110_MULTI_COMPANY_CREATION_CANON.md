## 1. Canon

AICompanyManager must allow the user to create multiple companies from the UI.

Each company has:
- company_id
- company_name
- business_domain
- robot_naming_rule
- company_policy
- delivery_policy
- organization_trees
- current_task
- delivery
- handoffs

## 2. UI Rule

Company creation is part of the settings screen.

The user can:
- add a company
- select a company
- edit selected company settings
- see changes reflected in the dashboard

## 3. System Use

The current app remains system-use.

No pricing or billing configuration is added.
