# ============================================================
# AICompanyManager CSV Template Prompt Copy Canon
# ============================================================

phase: Phase AF
status: csv-template-prompt-copy-canon-created

## 1. Canon

The CSV generation prompt must not be shown at the top of the main department task ledger screen.

It belongs to a separate screen/tab:

- CSV作成テンプレ

## 2. Required UX

CSV作成テンプレ screen contains:

- required columns
- optional columns
- CSV header
- sample CSV
- ChatGPT generation prompt
- one-click copy button

## 3. Copy Button

Button label:

- ChatGPT用プロンプトをコピー

## 4. Prompt Output Rule

The copied prompt instructs ChatGPT to output:

- CSV only
- no explanation
- no Markdown code fence
