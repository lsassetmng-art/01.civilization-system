# AICompanyManager robot nickname / combo add / organization tree UI canon

## Canon 1
Robots are selected from the Business-side robot pool/catalog, not arbitrary free-text robot creation.

## Canon 2
AICompanyManager is a system-use app, so robot allocation is unlimited.

## Canon 3
Allocation does not consume quantity.

## Canon 4
Assigned robots require a UI display name/internal company nickname.

## Canon 5
Display format is:
社内通称@役割

## Canon 6
Organization assignment is:
combo-box select -> role select -> internal nickname -> add

## Canon 7
A single organization may have multiple robot assignments.

## Canon 8
Organization tree should show:
department -> organization -> assigned robots
