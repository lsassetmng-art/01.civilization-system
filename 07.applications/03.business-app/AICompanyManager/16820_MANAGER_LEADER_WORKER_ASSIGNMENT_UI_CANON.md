# AICompanyManager Manager / Leader / Worker assignment UI canon

## Department
Department owns Manager robot setting.

## Section
Section owns Leader robot setting.

## Worker placement
Worker robots are placed under section detail.
One section may have multiple Worker robot assignments.

## Nickname
All assigned robots need an internal company nickname.

## Display
Display format:
社内通称@役割

## Allocation
AICompanyManager is system-use.
Robot allocation is unlimited.
No quantity is consumed.
