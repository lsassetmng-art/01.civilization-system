## 1. Canon

Work packet entry should minimize free text.

Use:

- dropdowns
- checkboxes
- presets
- templates
- file attachments
- generated summaries

Free text is limited to:

- short work title
- short supplemental note

## 2. Required Fields

- work packet name
- work type
- target type
- target category
- target name
- priority
- deadline
- target department
- delivery mode
- context pack
- expected output
- supplemental note

## 3. Bulk Department Delivery

Work packets support department-level bulk delivery.

A department assignment creates department inbox items.

Routing options:

- department manager
- department leader
- department all robots
- include child departments
- specified role only
