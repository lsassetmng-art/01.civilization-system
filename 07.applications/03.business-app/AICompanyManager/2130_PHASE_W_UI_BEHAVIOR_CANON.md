## 1. Required UI Behavior

The UI must support:
- add company
- edit selected company
- add organization tree inside selected company
- add organization unit inside selected tree
- select parent unit
- select AIWorker
- reflect saved settings in dashboard
- create multiple handoff packages

## 2. Dashboard

The dashboard must show:
- company list
- selected company overview
- role progress
- multiple organization trees
- task status
- delivery gate

## 3. Settings

The settings screen must show:
- company creation
- selected company settings
- organization tree creation
- organization unit creation

## 4. Human Review

Human still operates only at delivery.

Internal AI review remains automatic.
