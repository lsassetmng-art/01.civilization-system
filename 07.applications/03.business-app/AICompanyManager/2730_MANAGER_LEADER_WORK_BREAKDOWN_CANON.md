# ============================================================
# AICompanyManager Manager Leader Work Breakdown Canon
# ============================================================

phase: Phase AC
status: manager-leader-work-breakdown-canon-created

## 1. Flow

President:
- issues policy

Manager:
- creates initial broad breakdown table

Leader:
- expands Manager rows into deliverable units and task rows

Worker:
- executes task rows and creates concrete deliverables

## 2. Manager Broad Table

Manager creates rows such as:

- broad work area
- purpose
- target department
- expected deliverable group
- priority
- deadline
- policy reference

Example:
- 金融業務システム

## 3. Leader Expanded Table

Leader adds rows such as:

- deliverable unit
- task type
- worker role
- assigned robot
- required documents
- expected output
- status

Example:
- 窓口業務システム
- 入金管理
- 出金管理

## 4. Worker Deliverables

Worker outputs concrete deliverables such as:

- design document
- UI
- Java/code
- tests
- handoff one-block
- evidence
