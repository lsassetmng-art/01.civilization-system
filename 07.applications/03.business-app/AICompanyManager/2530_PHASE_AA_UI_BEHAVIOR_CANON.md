## 1. Dashboard

Company dashboard is the operation hub.

It shows:
- company summary
- role progress
- organization tree overview
- delivery gate
- navigation to company operation
- navigation to organization operation

## 2. Company Operation

The company operation screen shows:
- company selector
- add company form
- edit selected company form
- delete selected company action
- company rule file attachment area

## 3. Organization Operation

The organization operation screen shows:
- selected company
- add organization tree form
- add organization unit form
- edit/delete organization trees
- edit/delete organization units
- organization rule file attachment areas

## 4. Handoff

Handoff remains separate and continues to support additional file metadata.
