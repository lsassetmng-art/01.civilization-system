# AICompanyManager Reference File Supplemental Material Canon
# ============================================================

phase: Phase AE
status: reference-file-supplemental-material-canon-created

## 1. Reference File

Use:

- 参照ファイル

Meaning:
- materials that should be read before or during work
- materials used as the basis for decisions
- may include design documents, specifications, rules, error logs, test results, customer materials, and reference images

## 2. Supplemental Material

Use:

- 補足資料

Meaning:
- optional supporting materials attached to the task
- materials that help but are not the primary decision basis
- may include notes, screenshots, examples, rough drafts, prior notes, extra context, and supporting images

## 3. Department Task Ledger

Department task ledger rows contain:

- 成果物名
- タスク名
- 参照ファイル
- 補足資料
- 適用ルール
- 状態
- 納品物
- 引き継ぎ

Do not use "添付ファイル" as the primary user-facing label for these task-supporting materials.
