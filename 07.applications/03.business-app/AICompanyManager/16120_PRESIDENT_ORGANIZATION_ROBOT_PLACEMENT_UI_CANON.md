# AICompanyManager President / organization robot placement UI canon

## Canon 1: President is company-level
President robot is configured on AI企業設定.

## Canon 2: President receives company policy
Company business policy is instructed to President from AI企業設定.

## Canon 3: Organization robot placement is organization-owned
Robots are added to an organization from the organization card's add action.

## Canon 4: Robot source is Business registered robot catalog
Users select robots registered in Business tables.
The UI must not ask users to type an arbitrary robot name to create a new robot.

## Canon 5: Ai(ロボット) only
AICompanyManager placement target is Ai(ロボット).
Ai(人間) is not assignable as dispatched/placed AI worker.

## Canon 6: Local fallback is not canonical DB
The current localStorage fallback is only for UI actual-use smoke.
Persistent truth requires Business DB/API integration.
